# 定义设置回调函数

为了让过滤器能够保存设置，首先需要在 `hook_filter_info()` 的实现中为过滤器定义添加另一个回调。设置回调允许你定义表单元素，以便接收管理员输入的值。之后，Drupal 会在处理回调（以及准备回调，如果已定义）中提供这些值。

你需要添加一行代码，为过滤器指定一个设置回调函数。在 `dgd7_tip_filter_info()` 中创建的 `$filters['dgd7_tip']` 数组里加入以下内容。待稍后完成更多工作后，我会再次展示这个过滤器定义函数。

```
'settings callback' => '_dgd7_tip_settings',
```

当然，接下来你需要定义 `_dgd7_tip_settings()` 函数。该函数应返回表单元素，Drupal 会将这些元素嵌入到文本格式编辑页面中特定于过滤器的选项内。`filter_example.module` 中自然有一个示例，稍后你将根据自己的需求定义一个过滤器设置回调函数。
