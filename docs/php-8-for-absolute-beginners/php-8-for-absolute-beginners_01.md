# 引言

现代 Web 开发依赖于多种技术的成功集成。内容主要以 HTML 格式呈现。借助服务器端技术，您可以创建高度动态的 Web 应用程序。PHP 是交付基于浏览器的 Web 应用程序时使用最广泛的服务器端脚本语言。它是 Facebook、Flickr 和雅虎等在线巨头的支柱。

虽然还有其他用于 Web 应用程序开发的服务器端语言，但 PHP 是互联网的主力军。对于绝对的初学者来说，知道 PHP 是一种相对容易学习的语言应该会让人感到欣慰。使用少量 PHP 就能完成许多事情。此外，还有一个蓬勃发展的友好社区支持 PHP。您自己的 PHP 项目很容易获得帮助。

## 本书适合谁阅读

本书适合那些已经了解一些 HTML 和 CSS 的读者。它面向那些准备将 Web 开发技能提升到新水平的读者。您将学习使用 PHP 和 MySQL 动态生成 HTML 和 CSS。您将通过 PHP/MySQL 项目的实践体验，了解客户端脚本与服务器端脚本之间的区别。重点将放在快速上手并运行 PHP 上，但您也将在项目中使用一些 MySQL。到本书结束时，您将创建若干个由 PHP 驱动的项目，包括：

*   一个带有动态导航的个人作品集网站
*   一个动态图片库，用户可通过 HTML 表单上传图片
*   一个包含登录和管理模块的个人博客系统

在此过程中，您将熟悉诸如面向对象编程、设计模式、渐进增强和数据库设计等主题。您不会学到有关 PHP 的所有知识，但会打下良好的基础。

## 如何阅读本书

本书分为两个主要部分。第一部分将带您快速入门，使用 PHP 编写小型动态项目。您将接触到 PHP 相对较小的一个子集——足以为您开发入门级 Web 应用程序。第一部分还将教授您 PHP 的基本词汇。

第二部分是一个长篇幅的实践项目。您将从零开始，在指导下开发前面提到的个人博客系统。第二部分将向您展示如何使用 PHP 词汇来设计动态的、数据库驱动的 Web 应用程序。
