# PHP 快速入门清单

本章包含大量信息，不可能一次消化完。但希望前半部分已经让你对 PHP 的运行机制有了一个宏观的了解。后半部分则补充了许多关键细节，你可以在学习本书的过程中随时查阅。以下是部分要点回顾：

-   始终为 PHP 页面使用正确的文件扩展名，通常是 `.php`。
-   将所有 PHP 代码括在正确的标签内：`<?php` 和 `?>`。
-   避免使用短标签 `<?` 作为开始标记。使用 `<?php` 更为可靠。
-   建议在仅包含 PHP 代码的文件中省略 PHP 结束标签。
-   PHP 变量以 `$` 开头，后面跟一个字母或下划线。
-   选择有意义的变量名，并记住它们**区分大小写**。
-   使用注释来提醒自己脚本的功能。
-   记住，数字不需要引号，但字符串（文本）需要。
-   可以使用单引号或双引号，但必须配对使用。
-   使用反斜杠来转义字符串内部的同类引号。
-   要将相关项存储在一起，请使用数组。
-   使用条件语句（例如 `if` 和 `if . . . else`）进行决策。
-   使用循环简化重复性任务。
-   使用函数执行预设任务。
-   使用 `echo` 或 `print` 显示 PHP 输出。
-   使用 `print_r()` 检查数组的内容。
-   对于大多数错误消息，从指示的位置向前回溯。
-   保持微笑——并记住，PHP 并不难。
