本章讨论控制语句，这些指令旨在在程序中创建分支点以及重复执行语句块。换句话说，本章涵盖条件语句、循环语句和选择语句。

## `if` 条件语句

`if` 条件语句允许你根据某个条件执行不同的命令块。该条件语句有两种语法形式。通常使用以下选项。

```
if(condition){
# 命令
}
else{
# 命令
}
```

`if` 关键字后跟圆括号中的条件。然后，在花括号中放入命令。在命令块之后，放置 `else` 关键字，其后跟花括号中的其他命令。

我们来讨论一下条件语句是如何执行的。首先，检查 `if` 关键字后面的条件。如果条件为真，则执行条件后面的命令。如果条件为假，则执行 `else` 关键字后面的命令块。

`else` 块是可选的。可以省略不使用。在这种情况下，条件语句的语法如下所示。

```
if(condition){
# 命令
}
```

如果不使用 `else` 块，条件语句也会执行。首先检查条件；如果为真，则执行条件后面的命令。如果条件为假，则什么也不发生，并开始执行条件语句之后的指令。

> **注意：** 如果 `if` 块或 `else` 块只包含单个命令，可以省略花括号。但是，省略花括号会降低代码的可读性并导致错误。因此，始终使用花括号是合理的。

描述条件语句的另一种方式是不使用花括号，而是在条件后使用冒号、`else` 关键字，并在整个结构末尾使用 `endif` 语句（后跟分号）。在这种情况下，条件语句的通用语法如下所示。

```
if(condition):
# 命令
else:
# 命令
endif;
```

与之前的情况一样，`else` 块可以省略。如果省略，则按照以下模式声明条件语句。

```
if(condition):
# 命令
endif;
```

你可以使用嵌套的条件语句来依次检查多个条件。但更好的做法是使用带有 `elseif` 分支的条件语句变体。以下展示了这种条件语句的调用语法。

```
if(condition){
# 命令
}
elseif(condition){
# 命令
}
elseif(condition){
# 命令
}
else{
# 命令
}
```

这里使用了两个 `elseif` 分支，但原则上你可以使用任意多个 `elseif` 分支。接下来讨论一下条件语句的执行过程。首先检查 `if` 关键字后面的条件，如果该条件为真，则执行对应的命令分支。如果该条件为假，则检查 `elseif` 分支中的条件。如果该条件为真，则执行对应 `elseif` 分支中的命令。如果该条件为假，则继续检查下一个 `elseif` 分支中的条件，以此类推。如果所有检查过的条件都为假，则执行 `else` 分支中的命令。`else` 分支是可选的。

以下是使用 `elseif` 分支的条件语句的另一种语法形式。

```
if(condition):
# 命令
elseif(condition):
# 命令
elseif(condition):
# 命令
else:
# 命令
endif;
```

也就是说，不使用花括号，而是在每个分支的开头放置一个冒号，并且整个结构以 `endif` 关键字结束（后跟分号）。

## `while` 循环语句

循环语句使得重复执行指令或命令成为可能。PHP 中有多种循环语句。它们的作用是相同的（从能够解决相同任务、功能等效的角度来看）。

在所有的循环语句中，`while` 循环语句的语法或许是最简单的。它以 `while` 关键字开头，其后括号内放置一个条件，接着在花括号内跟一个命令块。其通用形式如下所示。

```
while(condition){
# commands
}
```

我们来讨论一下 `while` 循环语句是如何执行的。首先，检查条件。如果条件为真，则执行循环体中的命令。然后，再次检查条件。如果条件仍为真，则再次执行命令，并再次检查条件？如此往复。此过程持续进行，直到下一次条件检查结果为假。如果结果为假，则循环语句终止。

> **注意：** 如果循环体只包含一条命令，则可以省略花括号。但这并非良好的实践。

`while` 循环语句还有另一种形式，该形式不使用花括号，而是在条件后放置一个冒号，并且语句以 `endwhile` 指令（及分号）结束。

```
while(condition):
# commands
endwhile;
```

## `do-while` 循环语句

另一种循环语句是`do-while`语句。以下是该语句的语法。

```
do{
# 命令
}while(条件);
```

它以`do`关键字开头，后跟用花括号括起来的命令块，这些命令构成了**循环体**。接着是`while`关键字，并在括号中指定某个条件。我们来讨论一下`do-while`循环语句是如何执行的。首先，执行循环体中的命令。之后，检查`while`关键字后的条件。如果条件为真，则再次执行循环体中的命令，并再次检查条件。如果条件仍为真，则继续执行命令，依此类推。当下一次检查条件时，如果结果为`false`，则循环语句的执行终止。

![](img/613529_1_En_3_Figg_HTML.jpg) 信息图标。

**注意：** `do-while`语句与`while`语句的根本区别在于，在`while`语句中，先检查条件，然后执行命令。而在`do-while`语句中，则是先执行命令，然后检查条件。使用`while`语句时，可能会出现循环体中的命令一次都不执行的情况。这发生在第一次测试条件为假时。如果使用`do-while`语句，则循环体中的命令至少会执行一次。

| 详情 |
| --- |
| 如果在查找某个数字 *X* 的最小（大于 1 的）除数时，找到了某个值 *A*，这意味着原始数字可以表示为乘积 *X* = *A* · *B*，并且这里 *A* ≤ *B*。这就是为什么 *A*² ≤ *X*。这个关系给出了搜索除数范围的上限。 |
