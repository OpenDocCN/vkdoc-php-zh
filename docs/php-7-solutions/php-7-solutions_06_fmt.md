# 2. 准备使用 PHP 工作

既然你已决定使用 `PHP` 来丰富网页，就需要确保拥有完成本书后续内容所需的一切。虽然你可以在远程服务器上测试所有内容，但在本地计算机上测试 `PHP` 页面通常更方便。所有需要安装的软件都是免费的。在本章中，我将介绍 Windows 和 macOS 上的各种选项。Linux 系统通常会默认安装所需的组件。

本章涵盖：

- 检查你的网站是否支持 `PHP`
- 理解为何不应再使用 `PHP` 5
- 决定是否创建本地测试环境
- 在 Windows 和 macOS 中使用现成的集成包
- 决定存储 `PHP` 文件的位置
- 检查本地和远程服务器上的 `PHP` 配置
