# 第二章：变量

![image](img/frontdot.jpg)

变量用于存储数据，例如数字或字符串，以便在脚本中多次使用。

## 定义变量

变量以美元符号（`$`）开头，后跟一个*标识符*，即变量的名称。变量命名的一个常见惯例是，除第一个单词外，每个单词的首字母大写。

```
$myVar;
```

可以使用等号或赋值运算符（`=`）为变量赋值。此时变量被称为*已定义*或*已初始化*。

```
$myVar = 10;
```

一旦变量被定义，就可以通过引用变量名来使用它。例如，可以使用 `echo` 后跟变量名将变量的值打印到网页上。

```
echo $myVar; // "10"
```

请记住，变量名是区分大小写的。PHP 中的名称可以包含下划线和数字，但不能以数字开头。它们也不能包含空格或特殊字符，并且不能是保留关键字。

## 数据类型

PHP 是一种松散类型语言。这意味着变量可以存储的数据类型没有指定。相反，变量的数据类型会自动改变，以容纳其被赋予的值。

```
$myVar = 1;   // 整型
$myVar = 1.5; // 浮点型
```

此外，变量的值会根据其使用的上下文进行不同的求值。

```
// 浮点型被求值为字符串类型
echo $myVar; // "1.5"
```

由于这些隐式类型转换，了解变量的底层类型并不总是必要的。然而，理解 PHP 在后台处理的数据类型仍然很重要。以下表格列出了这九种类型。

| 数据类型 | 类别 | 描述 |
| --- | --- | --- |
| `int` | 标量 | 整数 |
| `float` | 标量 | 浮点数 |
| `bool` | 标量 | 布尔值 |
| `string` | 标量 | 字符序列 |
| `array` | 复合 | 值的集合 |
| `object` | 复合 | 用户自定义数据类型 |
| `resource` | 特殊 | 外部资源 |
| `callable` | 特殊 | 函数或方法 |
| `null` | 特殊 | 无值 |

### 整型

整数是整数。它们可以用十进制（基数为 10）、十六进制（基数为 16）、八进制（基数为 8）或二进制（基数为 2）表示法指定。十六进制数前加 `0x`，八进制数前加 `0`，二进制数前加 `0b`。

```
$myInt = 1234; // 十进制数
$myInt = 0b10; // 二进制数（十进制 2）
$myInt = 0123; // 八进制数（十进制 83）
$myInt = 0x1A; // 十六进制数（十进制 26）
```

PHP 中的整数始终是有符号的，因此可以存储正数和负数。整数的大小取决于系统的字长，因此在 32 位系统上，最大可存储值为 `2^(32-1)`。如果 PHP 遇到更大的值，则会被解释为 `float`。

### 浮点型

`float` 或浮点型可以存储实数。这些值可以使用十进制或科学计数法来赋值。

```
$myFloat = 1.234;
$myFloat = 3e2; // 3*10² = 300
```

`float` 的精度取决于平台。通常使用 64 位 IEEE 格式，该格式可以容纳大约 14 位十进制数字，最大十进制值为 `1.8x10³⁰⁸`。

### 布尔型

`bool` 类型可以存储一个布尔值，即只能为 `true` 或 `false` 的值。这些值使用 `true` 和 `false` 关键字指定。

```
$myBool = true;
```

### 空类型

不区分大小写的常量 `null` 用于表示没有值的变量。这样的变量被认为是特殊的 `null` 数据类型。

```
$myNull = null; // 变量被设置为 null
```

与其他值一样，`null` 值会根据其所在的上下文进行不同的求值。如果作为 `bool` 求值，则变为 `false`；如果作为数字，则变为零（`0`）；如果作为字符串，则变为空字符串（`""`）。

```
$myInt = $myNull + 0;      // 数值上下文 (0)
$myBool = $myNull == true; // 布尔上下文 (false)
echo $myNull;              // 字符串上下文 ("")
```

### 默认值

在 PHP 中，可以使用未赋值的变量。这种未定义的变量会自动创建，其值为 `null`。

```
echo $myUndefined; // 变量被设置为 null
```

尽管允许这种行为，但最好在使用变量之前对其进行定义，即使只是将其设置为 `null`。作为提醒，当使用未定义的变量时，PHP 会发出一个错误通知。根据 PHP 的错误报告设置，此消息可能会显示，也可能不会显示。

```
Notice: Undefined variable: myUndefined in
C:\xampp\htdocs\mypage.php on line 10
```
