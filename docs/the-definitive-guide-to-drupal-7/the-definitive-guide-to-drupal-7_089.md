# 自动化测试

完成上述工作后，下一步是使用自动化测试工具评估可能遗漏的问题。WebAim 的 WAVE（`wave.webaim.org`）是测试网站是否存在已知无障碍障碍的绝佳工具。他们还提供 Firefox 工具栏，对于评估需要用户登录的交互页面的影响尤为方便。功能无障碍评估器（`fae.cita.uiuc.edu`）可评估单个网页，注册后还能使用更多功能。像 Mozilla Firebug Ainspector（`code.google.com/p/ainspector/`）这类工具，可以扩展广受欢迎的 Firebug 工具，让你能够生成无障碍报告。

有不少工具可以帮助你更好地了解他人如何看待你的网站。谷歌的搜索机器人仍然是规模最大、信息最丰富的“盲人”网络用户，因此你为内容提供的语义信息越丰富，你的搜索引擎优化（SEO）效果就可能越好——同时也能提升网站的无障碍性。虽然屏幕阅读器的工作方式不同，但用 Lynx 浏览器查看你的网站，了解它以纯文本形式呈现的效果，仍是有价值的；详见 `en.wikipedia.org/wiki/Lynx_(web_browser)`。
