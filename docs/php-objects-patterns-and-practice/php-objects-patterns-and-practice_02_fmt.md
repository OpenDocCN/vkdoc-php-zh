# 致谢

一如既往，我在编写本版的过程中得到了许多人的支持。但同样一如既往地，我也必须回顾本书的起源。早在我们都惊叹于 PHP 5 的崭新可能性之时，我在布莱顿的一次演讲中尝试了本书的一些基本概念。感谢主持那次演讲的 Andy Budd，也感谢充满活力的布莱顿开发者社区。还要感谢 Jessey White-Cinis，他当时也在那次会议上，并让我联系上了 Apress 的 Martin Streicher。

这一次，Apress 团队再次提供了巨大的支持、反馈和鼓励。我很幸运能受益于如此专业的团队。

当我提出本版计划时，我轻率地承诺将更新所有代码以符合最新的风格指南。我非常幸运地邀请到我的朋友兼同事 Paul Tregoing 担任本版的技术审校。他不仅确保我履行了承诺，还深入代码本身，修正了许多许多标准违规之处，展现了超出职责范围的奉献精神！此外，本版还极大地受益于 Paul 的知识、见解和对细节的关注——非常感谢 Paul，Paul！

感谢并挚爱我的妻子 Louise，以及我们的孩子 Holly 和 Jake，感谢他们带来的许多宝贵分心时刻。感谢 Steven Metsker 的善意许可，让我在 PHP 中重新实现他在其著作《*Building Parsers in Java*》（Addison-Wesley Professional, 2001）中提出的解析器 API 的简化版本。

我写作时习惯听着音乐。在本书之前的版本中，我怀念了伟大的 DJ John Peel，这位地下音乐和折衷主义的捍卫者。本版的配乐主要由 BBC 广播 3 台的当代音乐节目《*Late Junction*》循环播放提供。感谢他们让一切保持古怪。
