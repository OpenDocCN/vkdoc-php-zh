## 多维数组

数组的元素也可以是数组。而这些内部数组的元素也可以是数组，依此类推。在这种情况下，必须使用多个索引或键来访问内部数组的元素：第一个键允许访问内部数组，第二个键允许访问该内部数组的元素，并且这个链条可以继续。如果是这样，我们通常称之为多维数组。但是，在 PHP 中，由于数组元素可以是不同类型，这个概念相当模糊。我们基于这样一个事实：多维数组是一个其元素中包含数组的数组。

从技术角度来看，创建多维数组并不复杂。在创建初始（外部）数组时，将一个或多个元素描述为数组即可。清单 4-7 显示了一个小例子。

## 连接数组

另一个实际的操作是数组的连接（加法）。为此，使用 `+` 运算符。规则如下：如果将数组加到另一个数组上，会得到一个由两个数组的元素组成的数组。如果合并的数组具有相同键的元素，则结果数组保留第一个操作数中的值（来自第一个数组的值）。清单 4-9 说明了这种情况。

## 比较数组

另一个重要的操作是比较数组是否相等或不相等。如果两个数组具有相同的元素集合（包括这些元素的键和值），则视为相等。但这里你需要考虑数组比较时涉及到的自动类型转换。也就是说，如果使用 `==` 和 `!=` 运算符比较数组，则会在考虑自动类型转换的情况下进行比较。如果你想在不涉及自动类型转换的情况下比较数组，则应使用 `===` 和 `!==` 运算符。此时仅考虑元素的值，而键在比较时无论如何都会涉及自动类型转换。列表 4-10 展示了一个数组比较的示例。

```
$A=["0"=>123,1=>true,2=>""];
$B=["123",10,0];
$C=[0=>123,"1"=>true,"2"=>""];
$D=[0=>123,1=>true,3=>""];
// 比较数组：
if($A==$B){
echo '$A==$B',"\n";
}
if($A!==$B){
echo '$A!==$B',"\n";
}
if($A===$C){
echo '$A===$C',"\n";
}
if($A!=$D){
echo '$A!=$D',"\n";
}
?>
列表 4-10
比较数组
```

程序执行结果如下：

```
程序输出（来自列表 4-10）

$A!==$B
$A===$C
$A!=$D
```

存在数组 `$A=["0"=>123,1=>true,2=>""]`，它与数组 `$B=["123",10,0]`、`$C=[0=>123,"1"=>true,"2"=>""]` 和 `$D=[0=>123,1=>true,3=>""]` 进行比较。结果表明 `$A==$B` 条件为真。原因如下：`$A` 数组的元素键为 `"0"`、`1` 和 `2`，对应的值分别为 `123`、`true` 和 `""`。`$B` 数组的元素键为 `0`、`1` 和 `2`（自动计算得出），对应元素的值分别为 `"123"`、`10` 和 `0`。键是匹配的，因为 `"0"` 被自动转换为 `0`。值 `"123"` 被转换为 `123`，非零数字 `10` 等价于 `true`，空字符串 `""` 被转换为零值 `0`。

![](img/613529_1_En_4_Figg_HTML.gif)一个铃铛符号的插图。**PHP 8 标准**

在 PHP 8 中，空字符串 `""` 不等于数字 `0`。因此，在 PHP 8 中，`$A==$B` 不为真。

如果不涉及自动类型转换，那么数组 `$A` 和 `$B` 的元素值是不同的，因此表达式 `$A!==$B` 为真。

在 `$C` 数组中，元素的值与 `$A` 数组相同。虽然 `$C` 数组中的元素键与 `$A` 数组不同，但由于自动类型转换，这些键被解释为相同。因此，条件 `$A===$C` 为真。最后，将 `$D` 数组与 `$A` 数组进行比较，只有一个键不同，这足以使条件 `$A!=$D` 为真。

数组 `$A=[1,2,3,4,5]` 被赋值给语句 `list($x,,$y,$z)=$A`。命令执行如下：`$A` 数组第一个元素的值 `1` 被存入变量 `$x`；`list()` 的下一个参数被省略，因此值为 `2` 的元素未被写入任何地方；值 `3` 被写入变量 `$y`；值 `4` 被写入变量 `$z`；值 `5` 未被写入任何地方，因为 `list()` 的参数列表中未指定对应的变量。

还有一些其他函数和工具在处理数组时也很有用。例如，`array_keys()` 函数返回一个包含作为参数传入的数组的键的数组。`array_combine()` 函数允许你基于两个数组创建一个新数组：一个数组定义键，另一个定义元素的值。另一个有用的函数是 `range()`，它创建一个包含指定范围内（函数参数）的元素值的数组。`array_pop()` 函数移除数组的最后一个元素，而 `array_push()` 函数在数组末尾添加一个（或多个）元素。`array_slice()` 函数允许你获取数组的一个切片（即从数组中提取一个子数组）。该函数的第一个参数是要切片的数组，第二个参数是切片起始元素的索引。如果该参数为负数，则从数组的最后一个元素开始计数（`-1` 表示最后一个元素，`-2` 表示倒数第二个元素，依此类推）。第三个参数指定切片中的元素数量。

`$K=range(3,5)` 命令创建一个从 `3` 到 `5` 的数字数组，包含两端（即 `[3,4,5]` 数组）。另一个数组由 `$A=["X"=>100,"Y"=>200,"Z"=>300]` 命令创建。`$B=array_keys($A)` 命令创建第三个数组。它是由 `$A` 数组的键组成的数组（即 `["X","Y","Z"]` 数组）。`$C=array_combine($K,$B)` 命令创建一个数组，其键由 `$K` 数组元素的值决定，而 `$C` 数组元素的值则由 `$B` 数组元素的值决定。

## 总结

*   数组是由一个共同名称连接的值的集合。这些值通常可以是不同的类型。
*   数组元素通过键或索引（在方括号内，位于数组名称之后）进行访问。键/索引可以是整数或字符串。
*   有几种方法可以创建数组。具体来说，您可以给数组元素赋值，使用 `array()` 函数（语法结构），或者在方括号中列出数组元素的值。如果未明确指定键/索引，则会自动计算。
*   您可以使用 `foreach` 循环语句来遍历数组的内容。它允许您访问数组元素的键和值。
*   数组元素本身也可以是数组。在这种情况下，内部数组的元素通过使用多个键来访问。
*   您可以对数组执行各种操作，包括赋值和比较相等/不相等。您还可以连接数组。
*   存在专门用于处理数组的函数。
