# 9. 上传文件

PHP 处理表单的能力不仅限于文本。它还可用于将文件上传到服务器。例如，你可以构建一个房地产网站，让客户上传其房产图片，或者为所有亲朋好友搭建一个上传假日照片的网站。然而，能做某事并不意味着你应该去做。允许他人向你的网站上传内容可能会让你面临各种问题。你需要确保图片尺寸合适、质量过关，并且不包含任何非法内容。你还必须确保上传的文件不包含恶意脚本。换句话说，你需要像保护自己的计算机一样小心地保护你的网站。

PHP 使得限制接受文件的类型和大小相对简单。但它无法检查内容的适用性。请仔细考虑安全措施，例如通过将上传表单放在受密码保护的区域中，将上传限制为已注册和受信任的用户。

在学会如何通过第 11 章和第 19 章的 PHP 限制页面访问之前，如果在公共网站上部署，请仅在受密码保护的目录中使用本章的 PHP 解决方案。大多数托管公司通过网站的控制面板提供简单的密码保护。

本章第一部分致力于理解文件上传的机制，这将有助于理解后面的代码。这是一个相当深入的章节，而不是快速解决方案的集合。但是到本章结束时，你将构建一个能够处理单文件和多文件上传的 PHP 类。然后，你只需编写几行代码，就可以在任何表单中使用该类。

你将学习以下内容：

- 理解 `$_FILES` 数组

- 限制上传文件的大小和类型

- 防止文件被覆盖

- 处理多文件上传
