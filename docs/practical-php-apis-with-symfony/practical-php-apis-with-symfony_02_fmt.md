# 第 1 节 前言

在本书中，我们将学习构建一个围绕操作（operations）的稳健且高效的 API 架构。学完本书，你将掌握使用 PHP 编程语言和 Symfony 框架构建面向操作的 API 所需的知识和必要技能。

本书旨在让你全面理解如何创建一个拥抱操作概念的单一 API 端点。我们将利用 Symfony 的强大功能来构建一个以操作为 API 设计焦点的环境。

请注意，本书假定你对 Symfony 框架有一定程度的熟悉。本书并非旨在作为 Symfony 的全面指南，而是专注于探索其构建面向操作的 API 的强大功能。

同样重要的是要明确，本书不涉及具体操作本身的开发。虽然我们可能会使用像`SendPayment`这样的操作示例进行说明，但主要目标是演示如何构建必要的代码来发现、管理和执行操作。

在后续章节中，我们将涵盖一系列关键的 Symfony 主题，例如依赖注入功能、安全认证以及使用 Symfony Messenger 处理的后台操作。
