编程是一个不断涌现新颖、高效且有趣事物的领域。因此，监控所有创新是必要的，但由于客观原因，这并不总是可行的。因此，“最低限度的策略”是理解并吸收构成某种技术基础的基本原则。这之所以重要，是因为理解基本概念能够让你快速、轻松地学习新方法并利用资源，就像将儿童积木的新元素添加到现有框架中一样。

本书讨论了与学习 *PHP 语言* 相关的最重要的主题和问题。除了实际的语法规则和结构之外，还存在如何在实践中使用语言这一方面。但要理解一门语言在实际中如何“运作”，了解它原则上能做什么至关重要。希望阅读本书并研究讨论过的示例后，你现在对 PHP 的潜力有了一个整体的认识。

还有一点需要强调。学习编程语言的原因多种多样。但主要的，或许是希望获得一份新职业，成为一名程序员。这意味着编程并不仅限于学习一门语言。每种语言都有其自身特点，但某些通用的编程原则对大多数语言是共通的。本书旨在帮助你，即使在直觉层面上，理解这些原则，并让你在职业道路上更容易地前进。当然，感谢你关注本书。
