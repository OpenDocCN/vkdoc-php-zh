## PHP：全景概览

乍看之下，PHP 代码可能看起来相当复杂，但一旦你理解了基础，你就会发现其结构非常简单。如果你使用过任何其他计算机语言（如 `JavaScript` 或 `jQuery`），你将会发现它们有很多共同之处。

每个 PHP 页面*必须*包含以下内容：

- 正确的文件扩展名，通常为 `.php`
- 包围每段 PHP 代码的 PHP 标记（如果文件仅包含 PHP 代码，则结束标记可省略）

一个典型的 PHP 页面会使用以下部分或全部元素：

- 变量，用作未知或变化值的占位符
- 数组，用于保存多个值
- 条件语句，用于做出决策
- 循环，用于执行重复任务
- 函数或对象，用于执行预设任务

让我们逐一快速浏览这些元素，从文件名和开始/结束标记开始。
