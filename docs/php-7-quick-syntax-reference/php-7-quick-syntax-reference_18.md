# 类型声明

类型声明允许函数声明其参数和返回值的预期类型。这使得 PHP 引擎能够强制使用指定的类型。

### 参数类型声明

早期的 PHP 版本完全依赖函数的正确文档，让开发者了解函数接受哪些参数。为了实现更健壮的函数，PHP 5 开始引入参数类型声明，允许指定函数参数的类型。类型声明的有效类型如表 18-1 所示，同时列出了这些类型被添加的 PHP 版本。

**表 18-1.** 类型声明

| 名称 | 描述 | 版本 |
| --- | --- | --- |
| 类名 | 参数必须是该类的对象或其子类。 | PHP 5.0 |
| 接口名 | 参数必须是实现该接口的对象。 | PHP 5.0 |
| `array` | 参数必须是数组。 | PHP 5.1 |
| `callable` | 参数必须可作为函数调用。 | PHP 5.4 |
| `bool` | 参数必须是布尔值。 | PHP 7.0 |
