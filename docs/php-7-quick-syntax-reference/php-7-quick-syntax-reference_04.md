# 变量

变量用于存储数据，例如数字或字符串，以便在脚本中多次使用。

### 定义变量

变量以美元符号 (`$`) 开头，后跟一个标识符，即变量的名称。变量常见的命名约定是，除了第一个单词外，每个单词的首字母大写。

```
$myVar;
```

可以使用等号或赋值运算符 (`=`) 为变量赋值。此时变量被定义或初始化。

```
$myVar = 10;
```

一旦变量被定义，就可以通过引用变量名来使用它。例如，可以使用 `echo` 后跟变量名将变量的值打印到网页上。

```
echo $myVar; // "10"
```

请记住，变量名是区分大小写的。PHP 中的名称可以包含下划线字符和数字，但不能以数字开头。它们也不能包含空格或特殊字符，并且不能是保留关键字。

### 数据类型

PHP 是一种松散类型的语言。这意味着不会指定变量可以存储的数据类型。相反，变量的数据类型会自动更改以容纳分配给它的值。

```
$myVar = 1;   // int 类型
$myVar = 1.5; // float 类型
```

此外，变量的值会根据其使用的上下文进行不同的求值。

```
// Float 类型被求值为 string 类型
echo $myVar; // "1.5"
```
