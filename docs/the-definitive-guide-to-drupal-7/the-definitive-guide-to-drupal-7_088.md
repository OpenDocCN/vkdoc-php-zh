# 对比度与色彩

了解人们感知世界的多种不同方式，能提升你为他们设计的能力。Drupal 文档提供了关于色彩运用及合理色彩对比需求的优质资源；详见 `drupal.org/node/464500`。

许多网站未能为内容提供足够的文本对比度，导致所有用户难以轻松阅读。有一些简单工具可以确保网站为低视力用户提供足够的对比度，例如 `webaim.org/resources/contrastchecker`。

色彩常被用于向用户传递信息，但并非所有人都能区分所有颜色。8% 的男性人口存在色觉障碍。如果你希望向用户传达重要信息，请将色彩与位置、邻近度以及图标等图形元素结合使用。这种对清晰度的额外关注，也有助于有学习障碍或认知障碍的用户解读和浏览你的网站。
