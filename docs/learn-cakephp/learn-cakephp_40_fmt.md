# 总结

在本章中，你品尝了一些“干货”。首先，你了解了如何生成和使用代码覆盖率，然后发现了一种生成测试固件的简便方法。你还学会了如何测试私有方法、视图、组件、辅助函数和插件。

这是我关于 CakePHP 与单元测试的书的结尾。希望你能充分利用本书中的大量信息，并帮助你成为一名更优秀的 Web 开发者。
