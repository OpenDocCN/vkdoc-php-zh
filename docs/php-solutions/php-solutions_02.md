# 使用示例文件

本书所需的所有文件均可从 Apress 网站下载，网址为 [`www.apress.com/9781484206362`](http://www.apress.com/9781484206362)。请确保选择《PHP 解决方案：动态网页设计轻松入门（第三版）》的下载链接。该版本的代码与前两版有所不同。

请按照第 2 章所述，搭建 PHP 开发环境。解压文件，将 `phpsols` 文件夹及其所有内容复制到 Web 服务器的文档根目录下。各章节的代码位于以章节名称命名的文件夹中：`ch01`、`ch02`，依此类推。请遵循每个 PHP 解决方案中的说明，将相关文件复制到网站根目录或指定的工作文件夹中。

当某一页面在章节内经历多次修改时，我会对不同的版本进行编号，例如：`index_01.php`、`index_02.php` 等。复制带有编号的文件时，请移除文件名中的下划线和编号，因此 `index_01.php` 应变为 `index.php`。如果你使用像 Dreamweaver 这类在移动文件时会提示更新链接的程序，请不要更新它们。文件中的链接设计为在目标文件夹中时能正确关联相应的图片和样式表。我这样做是为了让你能使用文件比较工具，将自己的文件与我的文件进行比对。

如果你没有文件比较工具，我强烈建议你安装一个。它能在你试图找出自己版本与我的版本之间的差异时，为你省去数小时的挠头苦思。在数十行代码中，一个遗漏的分号或拼错的变量可能很难被发现。Windows 用户可从 [`http://winmerge.org/`](http://winmerge.org/) 免费下载 WinMerge。我使用的是 Beyond Compare（[`www.scootersoftware.com`](http://www.scootersoftware.com/)），该工具现已支持 Windows、Mac OS X 和 Linux 平台。它虽非免费，但功能卓越且价格合理。Mac 上的 BBEdit 内置了文件比较工具。或者，你也可以使用 TextWrangler 中的文件比较功能，该软件可从 [`www.barebones.com/products/textwrangler/`](http://www.barebones.com/products/textwrangler/) 免费下载。
