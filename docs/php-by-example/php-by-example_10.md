数组通常是一组由共同名称统一的值。当有很多值，并且为每个值创建单独的变量不可能时，通常会使用数组。在这种情况下，整个值集使用一个通用名称，而特定值（数组元素）通过一个或多个索引来标识。在“经典”版本中，索引是整数。在 PHP 中，整数和字符串都可用作索引。在这种情况下，可能会使用术语“键”代替“索引”。具有字符串键的数组通常称为*关联数组*。

![](img/613529_1_En_4_Figa_HTML.jpg)一个 i 符号的图示。注意

关于术语，有很多可能的说法。例如，术语“列表”有时与“数组”一起使用。代替*关联数组*，使用术语“字典”或“表”。让我们将所有数组（整数索引和字符串键）都称为数组。*列表*是具有整数索引的数组。索引被解释为整数键的一种特殊情况。

接下来，让我们考虑一些创建和使用数组的案例。

| 详情 |
| --- |

```
## 遍历数组的循环

您已经研究过在`for`循环语句中使用索引来迭代数组元素的情况。这种方法的不便之处在于索引必须形成一个有序序列。但情况可能并非如此。因此，通常使用另一种循环语句。它被称为`foreach`。其语法如下。

```
