# 1. 使用 PHP

电子补充材料 本章的在线版本 (doi:[10.1007/978-1-4842-1922-5_1](http://dx.doi.org/10.1007/978-1-4842-1922-5_1)) 包含补充材料，仅供授权用户使用。

要开始使用 PHP 进行开发，请创建一个文件扩展名为`.php`的纯文本文件，并在你选择的编辑器中打开它——例如 Notepad、jEdit、Dreamweaver、NetBeans 或 PHPEclipse。这个 PHP 文件可以包含任何 HTML 以及 PHP 脚本代码。首先，为一个 HTML 5 网页文档输入以下最简标记。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>PHP Test</title>
</head>
<body>
    <p>Hello World</p>
</body>
</html>
```
