# 搜索产品目录

“你在找什么？” 无论在实体店还是电商商店中，这个问题都最常被问到。与其他优质的网络商店一样，你的 HatShop 也将允许访客搜索产品目录。

你将看到，通过将新组件集成到现有架构中，为运行中的网站添加新功能是多么简单。

在本章中，你将：

- 分析搜索产品目录的各种方式。

- 实现一个与 PostgreSQL 配合使用的自定义搜索引擎。

- 编写搜索功能的数据层和业务层。

- 使用 Smarty 组件化模板构建产品目录搜索功能的用户界面。

## 选择如何搜索产品目录

与往常一样，在开始编码之前，你需要思考几件事。在设计新功能时，你应该始终从最终用户的角度来分析该功能。

在视觉部分，你将使用一个文本框，访客可以在其中输入一个或多个词语进行搜索。在 HatShop 中，访客输入的词语将在产品描述和名称中
