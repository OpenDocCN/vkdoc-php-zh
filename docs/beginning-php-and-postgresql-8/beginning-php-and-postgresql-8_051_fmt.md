# 第 16 章 ■ 网络

## DNS、服务与服务器

如今，调查或排查网络问题通常需要收集大量与受影响客户端、服务器以及网络内部（例如协议、域名解析和 IP 寻址方案）相关的信息。`PHP` 提供了许多函数用于检索关于上述各个主题的大量信息，本节将对这些函数逐一介绍。

### DNS

`DNS` 使我们能够使用域名（例如 `example.com`）来代替其对应的不太友好的 IP 地址（例如 `192.0.34.166`）。域名及其对应的 IP 地址存储在遍布全球的域名服务器上，并可供查询。通常，一个域名会关联多种类型的记录，例如一种记录用于将 IP 地址映射到域名，另一种用于邮件路由，还有一种用于域名别名。网络管理员和开发者经常需要了解某个域名的各种 `DNS` 记录。本节将介绍一系列标准的 `PHP` 函数，它们能够挖掘出大量关于 `DNS` 记录的信息。

#### `checkdnsrr()`

`int checkdnsrr (string *host* [, string *type*])`

`checkdnsrr()` 函数根据提供的 `host` 值和可选的 `DNS` 资源记录类型检查 `DNS` 记录是否存在，如果找到任何记录则返回 `TRUE`，否则返回 `FALSE`。可能的记录类型包括以下几种：

- **A：** IPv4 地址记录。负责将主机名解析为 IPv4 地址。
- **AAAA：** IPv6 地址记录。负责将主机名解析为 IPv6 地址。
- **A6：** 一种用于表示 IPv6 地址的记录类型。旨在取代目前用于 IPv6 映射的 `AAAA` 记录。
- **ANY：** 查找任意类型的记录。
- **CNAME：** 规范名称记录。将别名映射到真实域名。
- **MX：** 邮件交换记录。确定主机的邮件服务器名称及相对优先级。此为默认设置。
- **NAPTR：** 命名权威指针。用于处理不符合 DNS 规范的名称，通过正则表达式重写规则将其解析到新域名。例如，`NAPTR` 可用于维护传统（前 DNS）服务。
- **NS：** 名称服务器记录。确定主机的名称服务器。
- **PTR：** 指针记录。用于将 IP 地址映射到主机名。
