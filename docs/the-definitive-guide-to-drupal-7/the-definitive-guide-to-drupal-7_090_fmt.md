# 模拟测试

拔掉鼠标，尝试仅通过键盘浏览你的网站，也是一种不错的方法。核心主题及之前推荐的主题都提供了跳过或直接跳转到网站导航链接的选项。这能让仅使用键盘的用户更轻松地浏览你的网站。更多信息，请参阅 AIM 网站（`webaim.org/techniques/keyboard/`）。

最强大、也是最受欢迎的屏幕阅读器当属 JAWS；然而，其他免费的屏幕阅读器也正在迅速流行起来。任何使用现代苹果电脑、iPad 或 iPhone 的用户，都内置了 VoiceOver 功能。Windows 用户可以免费下载 NVDA 软件应用程序（`www.nvda-project.org`），Linux 用户则可使用 ORCA（`live.gnome.org/Orca`）。

视力正常的人可以学习使用屏幕阅读器浏览网站，但这只是对盲人用户体验的一种近似模拟。即使是最近失明的人，其浏览网站的方式也与先天盲人不同。网页开发者使用屏幕阅读器浏览自己的网站会学到很多，但他们同时也带着关于信息架构如何运作的视觉认知。要真正理解如何最好地帮助盲人用户（或任何残障用户）浏览你的网站，始终鼓励来自这些残障人士本身的反馈是很有益的。
