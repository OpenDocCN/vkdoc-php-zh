# 引言

多年以来，开发网页和应用程序仍然是开发者们最着迷的事业之一。将一个简单的想法付诸开发，看着它逐渐成型，并想象用户将获得的体验，这其中的过程非常神奇。当然，作为 Web 开发者的编程体验也应该简单、友好且灵活——而本书所聚焦的 PHP 8 版本编程语言正具备所有这些特质。

PHP 的第一个版本由拉斯马斯·勒德尔夫在 1994 年创建，他主要在自己的主页上用它来追踪谁在浏览他的在线简历。1995 年，第一个公开版本以“个人主页工具”的名义发布。起初，PHP 只是一种简单的编程语言，拥有非常简单的解析引擎，只理解少量的工具和宏。

PHP 的使用量增长迅速。1996 年，大约有 15000 个网站是用 PHP 开发的；到 1997 年，这个数字达到了 50000 个。1999 年，大约有 100 万个网站是用 PHP 开发的。截至本书撰写之时，已有 7800 万个网站是用 PHP 开发的，这足以让你了解这种编程语言变得多么流行。为什么？很简单。因为多年以后它依然易于使用、用户友好且组织清晰，这有助于初学者轻松理解和运行 PHP 命令与函数。

我们希望能帮助 PHP 初学者和有志开发者探索这个 8 版本中添加的新特性，并了解使用它开发新网站和应用程序是多么简单、灵活且强大。你将找到关于如何安装和配置 PHP 8 的所有基本信息。所有基础的 PHP 概念，如数据类型、函数、正则表达式、表单处理与验证、会话、Cookie 和过滤器，都将通过大量示例进行介绍和演示。我们也会介绍面向对象编程以及相关类和对象的基础知识。

本书还提供了一些简单示例，展示如何将 PHP 8 与 MySQL（对开发者而言最强大、最常用的数据库之一）结合使用。你将学习如何使用 PHP 8 和 MySQL 开发一个完整的 Web 应用程序。你还会了解 PHP 框架以及它们为何如此重要。本书重点介绍众多 PHP 框架中的两个——Symfony 和 Laravel，并按照模型-视图-控制器架构模式来开发 Web 应用程序。

## 本书面向的读者

本书假设你具备一定的 Web 开发和数据库处理知识。本书主要面向希望学习如何使用 PHP 8，以及如何将其与 MySQL 和`Laravel`、`Symfony`等 PHP 框架结合使用的初级 Web 开发者。本书还假设你具备一些编程语言框架的知识，并了解如何以及何时在 PHP 中使用它们。

## 先决条件

本书中的示例使用 PHP 8 版本构建。我们还使用了适用于 Ubuntu Linux 22.04 版本的 MySQL DEB 安装包。作为测试工具，我们使用了网络上可获取的最新`Postman` API 客户端版，但你也可以自由使用任何你习惯的测试工具。最后，我们介绍并使用了两个不同的 PHP 框架，分别名为`Laravel`和`Symfony`，本书的示例中将需要用到它们。
