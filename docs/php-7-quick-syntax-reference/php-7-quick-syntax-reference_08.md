# 字符串转义与字符引用

## 字符转义表

| 字符 | 含义 | 字符 | 含义 |
|------|------|------|------|
| `\n` | 换行 | `\f` | 换页 |
| `\t` | 水平制表符 | `\$` | 美元符号 |
| `\v` | 垂直制表符 | `\'` | 单引号 |
| `\e` | 转义 | `\"` | 双引号 |
| `\r` | 回车 | `\\` | 反斜杠 |
| `\u{}` | Unicode 字符 | | |

例如，换行符在字符串中用转义字符`\n`表示。

```
$s = "Hello\nWorld";
```

请注意，此字符与 HTML 标签`<br>`不同，后者在网页上创建换行。

```
echo "HelloWorld";
```

当使用单引号或 nowdoc 定界符时，唯一生效的转义字符是反斜杠（`\\`）和单引号（`\'`）。仅在单引号前或字符串末尾需要转义反斜杠。

```
$s = 'It\'s'; // "It's"
```

PHP 7 引入了 Unicode 转义字符，能够在字符串中嵌入 UTF-8 编码的字符。此类字符以十六进制数形式指定，放在花括号内。数字最长可为六位，前导零可选。

```
echo "\u{00C2A9}"; // © (版权符号)
echo "\u{C2A9}";   // ©
```

### 字符引用

字符串中的字符可以通过在字符串变量后的方括号中指定所需字符的索引来引用，索引从零开始。这可用于访问和修改单个字符。

```
$s = 'Hello';
$s[0] = 'J';
echo $s; // "Jello"
```

函数`strlen`获取字符串参数的长度。例如，这可用于更改字符串的最后一个字符。

```
$s[strlen($s)-1] = 'y';
echo $s; // "Jelly"
```

### 字符串比较

比较两个字符串的方法只需使用相等运算符之一即可。这不会像某些其他语言那样比较内存地址。

```
$a = 'test';
$b = 'test';
$c = ($a === $b); // true
```

## 5. 数组

数组用于在单个变量中存储值的集合。PHP 中的数组由键值对组成。键可以是整数（数值数组）、字符串（关联数组）或两者的组合（混合数组）。值可以是任何数据类型。

### 数值数组

数值数组使用数字索引存储数组中的每个元素。数组使用`array`构造函数创建。该构造函数接受一个值列表，这些值被分配给数组的元素。

```
$a = array(1,2,3);
```

从 PHP 5.4 开始，可以使用更短的语法，用方括号替换`array`构造函数。

```
$a = [1,2,3];
```

数组创建后，可以通过在方括号中放置所需元素的索引来引用其元素。请注意，索引从零开始。

```
$a[0] = 1;
$a[1] = 2;
$a[2] = 3;
```

数组中的元素数量是自动管理的。向数组添加新元素只需为其赋值即可。

```
$a[3] = 4;
```

索引也可以省略，将值添加到数组末尾。如果变量尚未包含数组，此语法也会构造一个新数组。

```
$a[] = 5; // $a[4]
```

要检索数组中元素的值，需在方括号内指定该元素的索引。

```
echo "$a[0] $a[1] $a[2] $a[3]"; // "1 2 3 4"
```

### 关联数组

在关联数组中，键是字符串而不是数字索引，这为元素提供了名称而非数字。创建数组时，使用双箭头运算符（`=>`）来指示哪个键对应哪个值。

```
$b = array('one' => 'a', 'two' => 'b', 'three' => 'c');
```

关联数组中的元素使用元素名称进行引用。它们不能用数字索引引用。

```
$b['one']   = 'a';
$b['two']   = 'b';
$b['three'] = 'c';
echo $b['one'] . $b['two'] . $b['three']; // "abc"
```

双箭头运算符也可以与数值数组一起使用，以决定值放在哪个元素中。

```
$c = array(0 => 0, 1 => 1, 2 => 2);
```



并非所有键都必须指定。若某个键未被指定，则该值会被赋予一个比此前使用的最大整数键大 1 的键。

```
$e = array(5 => 5, 6);
```

### 混合数组

PHP 对关联数组和数字数组并不加以区分，因此这两类数组的元素可以混合在同一个数组中。

```
$d = array(0 => 1, 'foo' => 'bar');
```

只需确保使用相同的键来访问元素即可。

```
echo $d[0] . $d['foo']; // "1bar"
```

### 多维数组

多维数组是包含其他数组的数组。例如，可以按如下方式构造一个二维数组。

```
$a = array(array('00', '01'), array('10', '11'));
```

创建后，可以通过两组方括号修改其中的元素。

```
$a[0][0] = '00';
$a[0][1] = '01';
$a[1][0] = '10';
$a[1][1] = '11';
```

访问方式也相同。

```
echo $a[0][0] . $a[0][1] . $a[1][0] . $a[1][1];
```

可以为键赋予字符串名称，从而将其变为多维关联数组（也称为哈希表）。

```
$b = array('one' => array('00', '01'));
echo $b['one'][0] . $b['one'][1]; // "0001"
```

通过增加额外的方括号组，多维数组可以拥有超过两个维度。

```
$c[][][][] = "0000"; // 四维数组
```

## 6. 条件语句

条件语句用于根据不同的条件执行不同的代码块。

### If 语句

仅当括号内的条件计算结果为 `true` 时，`if` 语句才会执行。条件可以包含任何比较运算符和逻辑运算符。在 PHP 中，条件不一定是布尔表达式。

```
if ($x == 1) {
echo "x is 1";
}
```

为了测试其他条件，可以添加任意数量的 `elseif` 子句来扩展 `if` 语句。仅在之前所有条件均为 `false` 时，才会测试每个新增的条件。

```
elseif ($x == 2) {
echo "x is 2";
}
```

为了处理所有其他情况，可以在末尾添加一个 `else` 子句，当之前所有条件均为 `false` 时，该子句将被执行。

```
else {
echo "x is something else";
}
```

如果仅需有条件地执行单条语句，可以省略花括号。不过，通常建议始终包含花括号，因为它们能提高代码可读性。

```
if ($x == 1)
echo "x is 1";
elseif ($x == 2)
echo "x is 2";
else
echo "x is something else";
```

### Switch 语句

`switch` 语句会检查整数、浮点数或字符串是否与一系列 case 标签相等，然后将执行流程交由匹配的 case。该语句可以包含任意数量的 case 子句，并且可以使用 `default` 标签结尾来处理所有其他情况。

```
switch ($x)
{
case 1: echo "x is 1"; break;
case 2: echo "x is 2"; break;
default: echo "x is something else";
}
```

请注意，每个 case 标签后的语句没有用花括号括起来。相反，语句以 `break` 关键字结束，从而跳出 switch。如果没有 `break`，执行流程会继续向下一个 case 穿透。当多个 case 需要以相同方式求值时，这种特性很有用。

### 替代语法

PHP 为条件语句提供了一种替代语法。在此语法中，`if` 语句的开头花括号替换为冒号，结尾花括号被移除，并且最后一个结尾花括号被替换为 `endif` 关键字。

```
if ($x == 1):     echo "x is 1";
elseif ($x == 2): echo "x is 2";
else:             echo "x is something else";
endif;
```

类似地，`switch` 语句也有一种替代语法，它使用 `endswitch` 关键字来终止该语句。

```
switch ($x):
case 1:  echo "x is 1"; break;
case 2:  echo "x is 2"; break;
default: echo "x is something else";
endswitch;
```

对于较长的条件语句，替代语法通常更可取，因为这样更容易看出这些语句的结束位置。

### 混合模式

可以在代码块中间切换回 HTML 模式。这提供了另一种编写向网页输出文本的条件语句的方式。

```
This will show if $x is 1.

Otherwise this will show.

```

替代语法也可以这种方式使用，使代码更清晰。

```
This will show if $x is 1.

Otherwise this will show.

```

当输出 HTML 和文本，特别是较大的块时，通常更倾向于这种编码风格，因为它更容易区分 PHP 代码和网页上显示的 HTML 内容。

### 三元运算符

除了 `if` 和 `switch` 语句之外，还有三元运算符（`?:`）。此运算符可以替代单个 `if/else` 子句。该运算符接受三个表达式。如果第一个表达式求值为 `true`，则返回第二个表达式；如果为 `false`，则返回第三个表达式。

```
// 三元运算符表达式
$y = ($x == 1) ? 1 : 2;
```

在 PHP 中，此运算符既可用作表达式，也可用作语句。

```
// 三元运算符语句
($x == 1) ? $y = 1 : $y = 2;
```

编程术语*表达式*指的是能求值为一个值的代码，而*语句*则是一个以分号或结尾花括号结束的代码段。

## 7. 循环

PHP 中有四种循环结构。它们用于多次执行特定的代码块。与条件 `if` 语句一样，如果代码块中只有一条语句，则可以省略循环的花括号。

### While 循环

仅当 `while` 循环的条件为 true 时，它才会执行代码块。只要条件保持为 true，它就会持续循环。请注意，条件仅在每次迭代（循环）开始时进行检查。

```
$i = 0;
while ($i < 10) { echo $i++; } // 0-9
```

### Do-while 循环

`do-while` 循环的工作方式与 `while` 循环相同，区别在于它是在代码块执行之后才检查条件。因此，它至少会执行一次代码块。请记住，此循环以分号结尾。

```
$i = 0;
do { echo $i++; } while ($i < 10); // 0-9
```

### For 循环

`for` 循环用于按特定次数执行代码块。它使用三个参数。第一个参数初始化计数器，并且在循环开始前总是执行一次。第二个参数包含循环的条件，并且在每次迭代前进行检查。第三个参数包含计数器的增量，并且在每次迭代结束时执行。

```
for ($i = 0; $i < 10; $i++) { echo $i; } // 0-9
```

`for` 循环有多种变体，因为其中任何一个参数都可以省略。例如，如果省略第一个和第三个参数，它的行为方式与 `while` 循环相同。

```
for (;$i < 10;) { echo $i++; }
```

第一个和第三个参数也可以使用逗号运算符（`,`）拆分为多个语句。

```
for ($i = 0, $x = 9; $i < 10; $i++, $x--) {
echo $x; // 9-0
}
```

`sizeof` 函数用于获取数组中元素的数量。结合 `for` 循环，它可以用于遍历数字数组。

```
$a = array(1,2,3);
for($i = 0; $i < sizeof($a); $i++) {
echo $a[$i]; // "123"
}
```

如果不需要跟踪迭代次数，`foreach` 循环提供了更简洁的语法。此外，遍历关联数组也必须使用此循环。

### Foreach 循环

`foreach` 循环提供了一种遍历数组的简单方法。在每次迭代中，数组中的下一个元素会被赋值给指定的变量（迭代器），循环会持续执行，直到遍历完整个数组。

```
$a = array(1,2,3);
foreach ($a as $v) {
echo $v; // "123"
}
```

`foreach` 循环还有一种扩展形式，可以同时获取键的名称或索引：在迭代器之前添加一个键变量，后跟双箭头运算符（`=>`）。

```
$a = array('one' => 1, 'two' => 2, 'three' => 3);
foreach ($a as $k => $v) {
echo "$k => $v ";
}
```
