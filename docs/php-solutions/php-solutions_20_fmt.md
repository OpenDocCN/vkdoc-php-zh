# 5. 让表单生动起来

表单是 PHP 工作的核心。你可以使用表单登录受限页面、注册新用户、向在线商店下单、在数据库中输入和更新信息、发送反馈……这样的例子不胜枚举。所有这些用途背后的原理都是相同的，因此你在本章获得的知识将在大多数 PHP 应用中具有实用价值。为了演示如何处理表单信息，我将向你展示如何收集网站访问者的反馈，并将其发送到你的邮箱。

不幸的是，用户的输入可能会让你的网站暴露于恶意攻击之下。在接受数据之前，检查表单提交的数据非常重要。虽然 HTML5 表单元素在最新的浏览器中会验证用户输入，但你仍然需要在服务器端检查数据。HTML5 验证有助于合法用户避免提交带有错误的表单，但恶意用户可以轻松绕过在浏览器中执行的检查。服务器端验证并非可有可无，而是至关重要的。本章中的 PHP 解决方案将向你展示如何过滤或阻止任何可疑或危险的内容。防止恶意攻击并不需要耗费太多精力。同时，如果表单不完整或发现错误，保留用户输入并重新显示也是一个好主意。

这些解决方案将构建一个完整的邮件处理脚本，该脚本可以在不同的表单中重复使用，因此按顺序阅读它们很重要。

在本章中，你将了解以下内容：

- 理解用户输入如何从在线表单传输
- 在不丢失用户输入的情况下显示错误
- 验证用户输入
- 通过电子邮件发送用户输入
