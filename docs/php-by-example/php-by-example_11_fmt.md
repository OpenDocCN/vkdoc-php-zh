# 命令

`foreach`关键字后面跟着括号中的一个特殊“构造”。首先是数组名，然后是`as`关键字，接着是`key=>value`表达式。键和值是通过使用变量来实现的。循环语句的主体包含用花括号括起来的语句。

`foreach`循环语句的执行方式如下。在`foreach`语句中指定的数组中，元素被顺序迭代。对于数组的每个元素，执行循环主体中的命令。在`key=>value`语句中指定为键的变量获取元素键的值，而指定为值的变量获取元素的值。

另一种语法使用冒号代替左花括号，并使用`endforeach`关键字（以及分号）代替右花括号。

```
foreach(array as key=>value):
```
