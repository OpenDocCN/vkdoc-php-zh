本章介绍函数，它在任何编程语言中都扮演着重要角色。PHP 语言也不例外。您将学习如何创建函数、如何使用函数，以及 PHP 语言中函数有哪些独特之处。

## 创建函数

在程序中，经常需要多次执行相同的命令序列。将这样的命令写成一个独立的代码块，然后在需要时简单地放置一条指令来执行相应的命令块，是很方便的。概括来说，函数的使用就基于这一思想。

函数是一个可以通过其名称调用的命名代码块。换句话说，您组建一个命令块，为该命令块指定一个名称，每当需要执行该命令块时，只需指定其名称即可。通常，您会使用命令来处理某些值或数据。如果打算在函数调用期间处理这些值，则应在调用函数时传递一个参数（或多个参数）。传统上，函数参数写在函数名称后面的括号中。如果有多个参数，则用逗号分隔。

在调用函数之前，必须先声明（或创建）它。函数的描述意味着您必须定义在调用函数时执行哪些命令，以及对参数（如果有）执行哪些操作。那么，在 PHP 中如何定义函数呢？

函数声明以 `function` 关键字开头，后跟函数名称和括号中的参数列表。参数类似于变量，并通过调用函数时传递给函数的值来标识。如果函数没有参数，则在函数的描述中，函数名称后面放置一对空括号。调用函数时执行的命令被放置在一对花括号括起来的代码块中。以下是通用的函数声明模板。

```
function name(参数){
// 命令
}
```

函数体中的命令可以引用函数参数，且函数参数被描述为变量名（参数名称以 `$` 符号开头）。当调用函数并传递参数时，其主体中描述的命令将被执行。在这种情况下，在代码中放置参数的位置将使用传递给函数的实际值。

| 详细信息 |
| --- |
| 函数名称不区分大小写。换句话说，函数名称是使用大写字母还是小写字母都没有关系。例如，如果您有 `show()` 函数，那么它与 `SHOW()` 或 `Show()` 函数相同。还值得一提的是，传统上，函数名称后面会指定一对空括号。这通常是为了将函数名称与变量名称区分开来。如您所知，在 PHP 中，变量名称的显著特征是名称开头的 `$` 符号。尽管如此，这一传统仍然被保留。 |

## 函数的结果

函数可以返回一个结果。函数的结果是在函数执行完成后“保留下来”的值。如果函数返回一个结果，则调用函数的指令具有一个值，该值就是函数的结果。因此，例如，您可以将函数调用指令赋值给一个变量，或将其用作复杂语句中的操作数。为了理解包含函数调用指令的表达式的求值结果，您应该用表达式中相应函数返回的值替换这些指令。

为了在函数描述中表明函数返回结果，您需要使用`return`指令，后跟函数返回的值。

| 详情 |
| --- |
| 执行带有`return`语句的命令会终止函数的执行。函数代码中可以包含多个`return`指令（例如，在条件语句中），但只要其中一个被执行，函数执行就会终止。如果函数不返回结果，则可以使用`return`语句终止此类函数的执行，且无需指定返回值。通常，不返回结果的函数被解释为返回`null`引用。因此，原则上，对于此类函数，可以使用`return null`语句代替`return`指令（不带值）。 |

| 详情 |
| --- |
| 函数体中的变量称为*局部变量*。它们是在函数体中首次赋值的。局部变量在函数代码执行期间存在，并且仅在函数体内可用。由此得出两个重要结论。首先，仅在函数体中引用局部变量才有意义。其次，您可以在不同函数中使用相同名称的局部变量；这些是不同的变量，因为它们具有不同的作用域。函数声明中指定的参数具有“局部变量的效力”。它们仅在函数体内可用。如果在不同函数的描述中为参数指定了相同的名称，则它们是彼此独立的不同变量。 |

## PHP 函数定义与类型指定

`power()`函数用于计算一个数的幂。该函数有两个参数，在函数描述中用逗号分隔。第一个参数`$x`是要进行幂运算的数，幂次由第二个参数`$n`决定。

![](img/613529_1_En_5_Figf_HTML.jpg)包含字母'i'的信息图标。注意

尽管`power()`函数的第二个参数`$n`与`factorial()`函数的参数名称相同，但这些参数之间没有任何共同之处。同样的说明也适用于`power()`函数体中使用的局部变量`$s`。

在`power()`函数中，局部变量`$s`被初始化为初始值`1`。之后，在循环语句中执行`$s*=$x`命令，该命令将`$s`变量的当前值乘以`power()`函数的第一个参数`$x`。循环次数由第二个参数`$n`决定。最终，`$s`变量包含`$x`的`$n`次幂的值。`return $s`语句返回函数的结果。

程序中使用的另一个函数名为`test()`。它用于检查作为参数传递的数字（记为`$number`）是偶数还是奇数。如果数字是偶数，函数返回字符串`"even"`；如果数字是奇数，函数返回字符串`"odd"`。

函数代码的核心是一个条件语句，用于检查`$number%2==0`条件（`$number`参数除以`2`的余数为`0`）。如果条件为真，则通过`return "even"`命令返回结果；否则通过`return "odd"`返回结果。

你使用两个变量来测试函数的运行：值为`2`的`$z`和值为`10`的`$num`。通过`$res=power($z,$num)`命令，你用参数`$z`和`$num`调用`power()`函数，并将函数返回的结果存储到`$res`变量中。这就是`$z`变量的`$num`次幂的值。

保存在`$num`变量中的数字的阶乘通过`factorial($num)`指令计算。

最后，程序使用类似`"The number $num is ".test($num)."\n"`这样的表达式。这是通过连接`"The number $num is "`、`test($num)`和`"\n"`字符串得到的字符串。点号用作连接运算符，其中要连接的第二个字符串（`test($num)`指令）是`test()`函数返回的结果。

| 详细信息 |
| --- |
| PHP 解释器会根据源代码生成特殊的中间字节码。这个过程称为翻译。代码被翻译后，代码执行开始。在 PHP 中，翻译和代码执行是分离的。因此，函数不必在调用之前声明。换句话说，你可以先在程序中放置函数调用指令，然后再描述该函数。但不应过度使用这种风格，尤其是在实际使用 PHP 代码时，因为这会增加错误发生的可能性。 |

## 参数和结果的类型

在示例中描述函数时，假设传递给函数的值属于特定类型。这对于处理参数至关重要且被考虑在内。然而，无法保证在调用函数时，正确类型的值被作为参数传递。函数返回的结果类型也存在类似情况。

![](img/613529_1_En_5_Figg_HTML.jpg)包含字母'i'的信息图标。注意

参数类型或结果类型未被显式指定并不总是缺点；相反，这往往是优势。经常需要创建其参数可以是不同类型值的函数。但也存在相反的情况，即必须向函数传递严格定义类型的值。这完全取决于所解决问题的具体情况。

当然，你可以向函数中添加一些代码来检查传递的参数类型。但这并不总是方便。幸运的是，有一种更简单的方法来“限制”参数允许类型的范围。也就是说，你可以在函数描述中显式指定参数和结果的类型。这可以相当简单地完成。对于参数，类型在函数描述中的参数名称之前指定；函数结果类型在右括号之后用冒号分隔指定。这种情况下的函数描述模板如下。

```
function name(type argument, type argument,...): type{
// 命令
}
```

使用这种函数描述时，如果调用函数并传递与函数描述中指定的类型不同的参数，则会产生错误。

| 详细信息 |
| --- |
| PHP 有一个捕获和处理各种类型异常（错误）的系统。该系统允许你创建即使出现错误也能高效运行的程序。异常捕获将在后面讨论。 |

## 参数传递机制

来看看清单 5-4 中的程序。

```

```

```
输入：参数 100 和 200
输出：参数 200 和 100

```

你在程序中创建了带有两个参数的`swap()`函数。调用该函数时，它会显示传递给函数的参数。然后，你尝试更改参数的值（第一个参数被赋值为第二个参数的值，第二个参数被赋值为第一个参数的值），并再次检查它们的值。

使用了两个变量，`$A`和`$B`，其值分别为`100`和`200`。首先，检查这些变量的值。然后，将变量作为参数传递给`swap()`函数（使用`swap($A,$B)`命令）。接着，再次检查这些变量的值。

程序的输出是什么？正如你所见，`$A`和`$B`变量没有改变它们的值。然而，在`swap()`函数体内对参数的第二次检查显示，参数的值被交换了。这到底意味着什么，又该如何理解这种情况？其实，一切都非常简单。你应该考虑到，当参数传递给函数时，函数获取的是传递给函数的变量的副本。换句话说，在执行`swap($A,$B)`命令时，传递给`swap()`函数的不是`$A`和`$B`变量本身，而是它们的副本。因此，在执行`swap()`函数的代码时，变量的副本交换了值。当函数执行完毕，参数的副本会从内存中移除，而`$A`和`$B`变量仍然保持它们先前的值。这种将参数传递给函数的机制是默认的，称为按值传递参数。除了按值传递参数，还可以按引用传递参数。在这种情况下，传递给函数的是参数的“原件”而不是副本。如果你想按引用传递参数，应该在函数声明中的参数名称前加上`&`指令。对先前程序的修改版，但这次是按引用将参数传递给`swap()`函数，如清单 5-5 所示。

## 参数默认值

你可以为函数的参数设置默认值。当调用函数时，如果你没有为某个参数指定值，就会使用该默认值。

## 可变数量参数

PHP 的一个特性是：在调用函数时，你可以传入比函数声明中指定数量更多的参数，而不会产生任何灾难性后果。例如，如果一个函数被描述为有两个参数，而你在调用该函数时传入了五个参数，那么最后三个参数将被忽略。

函数的这个特性可以用来创建接受可变数量参数的函数。这意味着，一个函数可以用任意数量的参数调用，并且所有这些参数都会在函数内部被处理。

> **注意：** 你正在处理函数参数的默认值。这使得在调用函数时可以指定不同数量的参数（少于函数描述中指示的数量）。但在这种情况下，被处理的参数数量仅限于函数描述中指定的那些。与此相反，你希望无论调用函数时传入多少参数，所有参数都能被处理。

还值得一提的是，声明带有可变数量参数的函数有几种模式。你可以将其中一种称为“旧”模式，另一种称为“新”模式（自 PHP 5.6 起可用）。让我们来探讨一下这两种模式。

为了处理传递给函数的参数，需要以某种方式访问它们。此外，你还希望处理那些未在函数声明参数列表中提及的参数。你可以使用 `func_get_args()` 函数获取传递给函数的参数列表。`func_get_args()` 函数没有参数，它返回一个列表（数组），其中包含传递给调用 `func_get_args()` 函数的那个函数的所有参数。

| 详细信息 |
| --- |
| 函数 `func_num_args()` 和 `func_get_arg()` 也可能很有用。第一个函数返回传递的参数数量（指的是在其函数体内调用 `func_num_args()` 的那个函数的参数）。第二个函数返回具有特定索引（`func_get_arg()` 函数的参数）的参数值（调用 `func_get_arg()` 函数的那个函数的参数）。 |

## 匿名函数

在 PHP 中，可以定义没有名称的函数。这类函数通常被称为*匿名函数*。匿名函数的优雅之处和实际价值在于，它们可以作为值赋给变量。

| 详情 |

| --- |

| 匿名函数可以被表示为一个可调用的对象（这很接近事实），即可作为函数使用。如果这样一个对象（即匿名函数）被赋给一个变量，那么该变量可以像函数名一样被使用。 |

匿名函数的描述方式与普通函数类似，但有两点重要区别。首先，函数名不指定（在`function`关键字后直接跟参数列表）。其次，这样的函数通常会立即赋给一个变量。创建匿名函数的一般模板如下所示。

```
$variable=function(arguments){
# commands
};
```

实际使用效果如清单 5-11 所示。

```

for($s=0,$k=1;$k<=$n;$k++){

// 调用函数：
```

它定义了一个匿名函数，该函数基于参数`$n`，计算并返回从`1`到`$n`的自然数平方和。一旦该匿名函数被赋给`$A`（注意赋值语句末尾的分号），`$A`就可以像函数一样被使用——特别是，它可以被调用。例如，表达式`$A($n)`的值（变量`$n`的值为`4`）是从`1`到`4`的平方和（即`30`）。接着，执行`$B=$A`命令将`$A`赋给变量`$B`后，`$B`变量也“变成了函数”，且与`$A`相同。然后，`$A`变量被赋予一个新的匿名函数，定义如下。

```
function($x,$y){

}
```

这次是一个有两个参数的函数，返回两个参数的和。因此，表达式`$A($x,$y)`的值（变量`$x`和`$y`的值分别为`5`和`10`）是`15`。

## 命名参数

调用函数时，通常按照函数声明中指定的顺序传递参数。这种参数传递方式称为位置传递，或称按位置传递参数。然而，PHP 8 引入了按名称传递参数的能力。在这种情况下，调用函数时，参数（全部或部分）由名称和值定义。函数本身的描述方式与往常相同。

| 详情 |
| --- |
| 如果在函数声明中指定了某个参数，那么在调用函数时，该参数的名称不带`$`符号。参数名称后跟冒号，然后是参数值。例如，如果在`func()`函数声明中使用了`$arg`参数，那么在调用函数时，使用`func(arg: value)`指令来按名称传递参数。 |

```
?>

```


现在，对于`show()`函数，参数`$first`、`$second`和`$third`分别具有默认值`1`、`2`和`3`。因此，当使用`show(100,third:300)`命令调用该函数时，`$first`参数被设置为`100`，`$second`参数使用默认值`2`，而`$third`参数被设置为`300`。

如果使用`show(second:200)`命令，则`$second`参数被设置为`200`，而`$first`和`$third`参数分别使用默认值`1`和`3`。

## 总结

- 函数是一个可以通过名称调用的命名代码块。函数声明以`function`关键字开头，后跟函数的参数（在圆括号内）。然后，命令被包含在花括号中。当函数被调用时，这些命令会被执行。

- 函数可以返回一个结果。函数的返回值在`return`指令之后指定，该指令会终止函数的执行。

- 对于函数，你可以显式指定参数的类型和结果的类型。

- 默认情况下，函数的参数是通过值传递的（会创建作为函数参数指定的变量的副本）。要通过引用传递参数，在函数声明中，需要在相应参数名称前加上`&`引用符号。

- 参数可以有默认值（通过在函数声明中参数名称后使用赋值运算符指定）。如果在调用函数时没有为参数指定值，则会使用默认值。

- 你可以创建具有任意数量参数的函数。在这种情况下，整个参数集被标识为一个参数，在函数声明中该参数前加有省略号。该参数被视为一个数组。

- 当函数调用自身时（但通常使用不同的参数），可以使用递归来定义函数。

- 函数（语法结构）`eval()`允许你执行作为参数传递给该函数的字符串中的命令。

- 可以创建没有名称的匿名函数。匿名函数被赋值给一个变量；之后，可以像调用函数一样调用该变量。

- 从 PHP 8 开始，在调用函数时可以按名称传递参数。在这种情况下，你需要指定参数的名称（不带`$`符号），并在调用函数时用冒号分隔指定参数的值。
