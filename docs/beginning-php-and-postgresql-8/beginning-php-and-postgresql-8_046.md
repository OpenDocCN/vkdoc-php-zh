# 处理文件上传

尽管大多数人倾向于将互联网仅等同于网页，但 HTTP 协议实际上促进了任何类型文件的传输，例如 Microsoft Office 文档、PDF、可执行文件、MPEG、压缩文件以及各种其他文件类型。虽然从历史上看，FTP 一直是向服务器上传文件的标准方式，但通过基于 Web 的界面进行此类文件传输正变得越来越普遍。在本章中，你将全面了解 PHP 的文件上传处理能力。具体来说，本章主题包括：

- PHP 的文件上传配置指令
- 用于处理文件上传数据的 PHP `$_FILES` 超全局数组
- PHP 内置的文件上传函数：`is_uploaded_file()` 和 `move_uploaded_file()`
