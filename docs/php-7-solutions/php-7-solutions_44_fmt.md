# 3. 上传文件

在构建文件上传类之前，最好先创建一个简单的文件上传脚本，以确保你的系统能正确处理上传。

## 3.1. 将临时文件移动到上传文件夹

已上传文件的临时版本仅短暂存在。如果你不对该文件进行任何操作，它将被立即丢弃。你需要告诉 PHP 将其移动到哪里以及如何命名。这通过 `move_uploaded_file()` 函数完成，该函数接受以下两个参数：

*   临时文件的名称

*   文件新位置的完整路径，包括文件名本身

获取临时文件本身的名字很简单：它作为 `tmp_name` 存储在 `$_FILES` 数组中。由于第二个参数需要完整路径，这给了你重命名文件的机会。目前，我们保持简单，使用原始文件名，它作为 `name` 存储在 `$_FILES` 数组中。
