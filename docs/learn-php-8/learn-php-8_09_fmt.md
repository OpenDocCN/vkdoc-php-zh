# 第 9 章：多功能接口

完成本章学习后，学生将能够：

- 创建一个完整的、能够实现删除、更新和插入数据的 PHP 应用程序
- 使用 CSS 为完成的应用程序打造专业外观
- 使用 JavaScript 接收和操作来自其他程序的数据
- 保护应用程序内所有需要用户 ID/密码的程序
- 使用来自 JSON 对象的值填充 HTML 对象
