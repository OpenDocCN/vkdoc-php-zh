# 关于本书

本书完全致力于 PHP 语言，重点阐述其必要性及其特性。它探讨了语言的主要结构，以及在 PHP 中使用的方法和机制。本书非常重视示例。书中讨论的主题范围足以让你编写出高效的 PHP 代码。本书主体部分由十二章组成。

- **第 1 章**包含简单的程序，讨论了如何使用软件，并分析了在那里创建 PHP 程序的一般原则。

- **第 2 章**讨论变量和数据类型。你将学习如何创建变量、对它们执行操作以及使用不同类型的数据。

- **第 3 章**专门讲述控制语句。你将了解条件语句、选择语句和循环语句，以及 `goto` 指令。所有这些对于创建高效的 PHP 程序至关重要。

- **第 4 章**包含有关数组的信息。在 PHP 中，数组具有几个独特的功能，将是研究的主题。本章讨论了一维和多维数组，描述了集合遍历语句，并概述了对数组执行的基本操作。

- **第 5 章**描述了函数。你将学习如何创建函数，讨论参数传递的机制，考虑如何返回结果，学习如何为参数设置默认值，以及如何创建具有任意数量参数的函数。你将考虑按名称传递参数的方式。本章还重点介绍了递归、匿名函数以及与函数相关的其他一些主题。

- **第 6 章**涉及链接、常量、全局和静态变量、文件处理以及处理多行文本的方法。

- **第 7 章**讨论面向对象编程（OOP）的原则，并考虑 PHP 如何实现这些原则。你将学习如何创建类和对象，以及如何处理方法。你还将了解构造函数、析构函数和静态类成员。本章处理了对象复制的问题，并说明了如何使用私有字段和方法。本章还描述了一些特殊方法。

- **第 8 章**专门讲述一个重要的 OOP 机制：继承。你将学习如何创建子类以及如何重写方法。你将考虑在继承的背景下，构造函数、私有和受保护类成员的特性。本章还讨论了方法的虚拟性，描述了多级继承，并包含了一些其他信息。

- **第 9 章**描述了抽象类和接口（包括它们的实现和继承）。你将了解什么是 trait，接口如何控制对象的类型，什么是命名空间以及如何使用它。

- **第 10 章**错误和异常处理的原则在第 10 章中讨论。你将熟悉主要的异常类，并学习异常是如何产生的。此外，本章还描述了为用户自定义异常创建类的方法。还涵盖了与错误和异常处理相关的其他一些问题。

- **第 11 章**专门讲述生成器和迭代器。你将学习在不同情况下使用和应用生成器函数。你还将了解什么是迭代器以及如何创建它们。
