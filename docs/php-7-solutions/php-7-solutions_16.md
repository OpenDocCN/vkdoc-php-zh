# 理解 PHP 类与对象

类是**面向对象编程** (OOP）的基础构建模块，这种编程方法旨在提高代码的可复用性和可维护性。PHP 对 OOP 提供了广泛支持，并且新特性也经常以面向对象的方式实现。

**对象**是一种复杂的数据类型，能够存储和操作值。**类**则是定义对象特征的代码，可视为创建对象的蓝图。在 PHP 众多内置类中，特别值得一提的是 `DateTime` 和 `DateTimeZone` 类，它们用于处理日期和时区。

创建对象需使用 `new` 关键字和类名，示例如下：

```
$now = new DateTime();
```

这段代码创建了 `DateTime` 类的一个**实例**，并将其存储在名为 `$now` 的 `DateTime` 对象中。该对象不仅知道创建时的日期和时间，还了解 Web 服务器使用的时区。大多数类都拥有属性和方法，它们类似于变量和函数，但专属于某个特定的类实例。例如，可以使用 `DateTime` 类的方法更改某些值，如月份、年份或时区。`DateTime` 对象还能执行日期计算，而此类计算若用普通函数实现会复杂得多。

访问对象的属性和方法需要使用 `->` 运算符（一个连字符后跟一个大于号）。要重置 `DateTime` 对象的时区，需将 `DateTimeZone` 对象作为参数传递给 `setTimezone()` 方法，示例如下：

```
$westcoast = new DateTimeZone('America/Los_Angeles');
$now->setTimezone($westcoast);
```

无论 Web 服务器位于何处，上述代码都会将 `$now` 重置为洛杉矶的当前日期和时间，并自动调整夏令时。

访问对象的属性同样使用 `->` 运算符：

```
$someObject->propertyName
```

> **提示**  
> 对象、属性和方法的概念可能难以理解。你只需要掌握如何使用 `new` 关键字实例化对象，以及如何使用 `->` 运算符访问属性和方法即可。
