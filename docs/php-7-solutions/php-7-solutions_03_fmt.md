# PHP 的成长历程

PHP 起步于 1995 年，当时的目标相当有限。它最初被称为个人主页工具（`PHP Tools`）。其主要目标之一是通过收集在线表单的信息并在网页上显示，从而创建留言簿。三年之内，人们决定从名称中删除“个人主页”，因为它听起来像是业余爱好者的东西，无法充分体现此后添加的一系列复杂功能。这就留下了首字母缩写 PHP 应该代表什么的问题。最终，它被命名为 PHP 超文本预处理器——一个相当不雅的选择，因此大多数人干脆直接称其为 `PHP`。

多年来，PHP 不断发展，不断添加新功能。除了被一些大型网站使用之外，它还是驱动着 Laravel（`https://laravel.com/`）、Drupal（`http://drupal.org/`）和 WordPress（`http://wordpress.org/`）等极其流行的框架和内容管理系统（`CMSs`）的语言。

该语言的一大吸引力在于它始终不忘初心。尽管它支持复杂的面向对象编程，但你无需深入复杂的理论就可以开始使用它。PHP 的原始创建者拉斯马斯·勒德尔夫曾将其描述为“一种非常对程序员友好的脚本语言，适合几乎没有或完全没有编程经验的人，也适合需要快速完成工作的经验丰富的 Web 开发人员。”你可以立即开始编写有用的脚本，同时自信地知道你正在使用一种能够开发工业级应用程序的技术。

### 注意

尽管 PHP 7 大部分向后兼容 PHP 5，但本书中的许多代码使用了 PHP 7 的新特性。PHP 6 从未正式发布。
