# 函数

函数是一段可重用的代码块，仅在调用时执行。

## 定义函数

要创建函数，需要使用 `function` 关键字，后跟函数名、一对圆括号和一个代码块。函数的命名约定与变量相同——使用描述性名称，除首单词外每个单词首字母大写。

```
function myFunc()
{
  echo "Hello World";
}
```

函数代码块可以包含任何有效的 PHP 代码，包括其他函数定义。

## 调用函数

函数一旦定义，就可以通过输入其名称后跟一对圆括号从页面上的任何位置调用（调用）。函数名不区分大小写，但最好使用与其定义时相同的大小写。

```
myFunc(); // "Hello World"
```

即使函数定义出现在脚本文件的更下方，也可以调用该函数。

```
foo(); // 没问题
function foo() {}
```

但有一个例外：当函数仅在满足特定条件时才被定义时，必须在该条件代码执行之后才能调用该函数。

```
bar(); // 错误
if (true) { function bar() {} }
bar(); // 没问题
```

## 函数参数

函数名后的圆括号用于向函数传递参数。为此，必须先在函数定义中以逗号分隔的变量列表形式指定相应的参数。然后即可在函数中使用这些参数。

```
function myFunc($x,$y)
{
  echo $x . $y;
}
```

指定参数后，即可使用相同数量的参数调用函数。

```
myFunc('Hello', ' World'); // "Hello World"
```

准确地说，*形参*出现在函数定义中，而*实参*出现在函数调用中。然而，这两个术语有时可以互换使用。

### 默认参数

可以通过在参数列表中为参数赋值来指定默认值。这样，如果在调用函数时未指定该参数，则会使用默认值。为了使此功能按预期工作，带有默认值的参数必须声明在不带默认值的参数的右侧。

```
function myFunc($x, $y = " Earth")
{
  echo $x . $y;
}

myFunc("Hello"); // "Hello Earth"
```

### 可变参数列表

调用函数时传递的参数不能少于其声明中指定的数量，但可以多于声明的数量。这允许传递可变数量的参数，然后可以通过几个内置函数来访问这些参数。要一次获取一个参数，可以使用 `func_get_arg` 函数。该函数接受一个参数，即要返回的参数的索引（从零开始）。

```
function myFunc()
{
  $x = func_get_arg(0);    // 获取指定参数
  $y = func_get_arg(1);
  $z = func_get_arg(2);
  echo $x . $y . $z;
}

myFunc('Fee', 'Fi', 'Fo'); // "FeeFiFo"
```

还有两个与参数列表相关的函数。`func_num_args` 函数用于获取传递的参数数量，`func_get_args` 则返回一个包含所有参数的数组。

```
function myFunc()
{
  $num  = func_num_args();
  $args = func_get_args();
  for ($i = 0; $i < $num; $i++)
    echo $args[$i] . ' ';
}

myFunc('Fee', 'Fi', 'Fo'); // "Fee Fi Fo"
```

## Return 语句

`Return` 是一个跳转语句，它会导致函数结束执行并返回到调用它的位置。

```
function myFunc()
{
  return; // 退出函数
}
```

它可以选择返回一个值，在这种情况下，函数调用将求值为该值。

```
function myFunc()
{
  return 'Hello'; // 退出函数并返回值
}

echo myFunc();    // "Hello"
```

没有返回值的函数将被求值为 `null`。

## 作用域与生命周期

通常，PHP 变量的作用域从其声明处开始，一直持续到页面结束。然而，在函数内部会引入一个局部函数作用域。在函数内部使用的任何变量默认都限于此局部作用域。一旦函数作用域结束，局部变量就会被销毁。

```
$x = 'Hello';    // 全局变量

function myFunc()
{
  $y = ' World'; // 局部变量
}
```

在 PHP 中，尝试从函数内部访问全局变量是行不通的，反而会创建一个新的局部变量。为了使全局变量可访问，必须通过使用 `global` 关键字声明该变量，将其作用域扩展到函数内部。

```
$x = 'Hello';     // 全局 $x

function myFunc()
{
  global $x;      // 在此函数中使用全局 $x
  $x .= ' World'; // 修改全局 $x
}

myFunc();
echo $x;          // "Hello World"
```

另一种访问全局作用域变量的方法是使用预定义的 `$GLOBALS` 数组。注意，变量需要作为不带美元符号的字符串来指定。

```
function myFunc()
{
  $GLOBALS['x'] .= ' World'; // 修改全局 $x
}
```

与许多其他语言不同，控制结构（例如循环和条件语句）没有自己的变量作用域。因此，在此类代码块中定义的变量不会在代码块结束时被销毁。

```
if(true)
{
  $x = 10; // 全局 $x
}

echo $x;   // "10"
```

除了全局变量和局部变量之外，PHP 还有属性变量，这将在下一章中介绍。

## 匿名函数

PHP 5.3 引入了匿名函数，允许将函数作为参数传递并赋值给变量。匿名函数的定义方式与常规函数类似，只是没有指定名称。该函数可以使用普通的赋值语法（包括分号）赋值给变量。

```
$say = function($name)
{
  echo "Hello " . $name;
};

$say("World"); // "Hello World"
```

匿名函数可以继承其定义所在作用域中的变量。此类继承的变量通过在函数头部使用 `use` 子句来指定。

```
$x = 1;
$y = 2;

$callback = function($z) use ($x, $y)
{
  return $x + $y + $z;
};

echo $callback(3); // "6"
```

## 函数重载

与许多其他语言不同，PHP 不允许使用不同参数多次定义一个函数。这个特性称为函数重载，其优点是允许一个函数透明地处理各种参数，对用户来说没有区别。

尽管 PHP 没有函数重载，但可以实现类似的行为。由于 PHP 没有强类型，函数参数已经可以接受任何数据类型。这种松散类型——配合默认参数值和可变参数列表——使得复制函数重载的效果成为可能。

```php
// 打印可变数量的参数
function myprint()
{
  $a = func_get_args();
  foreach ($a as $v) { echo $v; }
}

myprint(1,2,3); // "123"
```

## 内置函数

PHP 提供了大量随时可用的内置函数，例如字符串和数组处理函数。其他函数则取决于 PHP 编译时集成的扩展，比如用于与 MySQL 数据库通信的 MySQL 扩展。关于 PHP 内置函数的完整参考，请参阅 `PHP 函数参考`¹。
