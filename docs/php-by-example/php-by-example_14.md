## 常量

常量与变量有根本的不同，在于常量的值不能被更改。`define()` 函数（更准确地说，是一种语法结构）通常用于向程序添加常量。该函数的第一个参数是一个包含常量名称的字符串。第二个参数指定常量的值。常量名称的开头不使用 `$` 符号。按照普遍接受的约定，常量名称由大写字母组成（也就是说，这不是必须的，但出于良好的编程习惯应当如此）。例如，命令 `define("PI",3.141592)` 定义了值为 `3.141592` 的常量 `PI`。执行该命令后，常量 `PI` 的值就不能再更改了。

常量在创建后不能被删除。要检查一个常量是否存在，可以使用 `defined()` 函数，其参数是一个包含常量名称的字符串；例如，`defined("PI")`。结果根据该常量是否被定义，返回 `true` 或 `false`。

还有一种向程序添加常量的方法。在这种情况下，使用关键字 `const`，其后指定常量的名称（名称开头不带 `$` 符号），并通过赋值运算符指定常量的值。例如，值为 `10` 的常量 `TEN` 可以通过命令 `const TEN=10` 来定义。

这两种创建常量的方法之间的根本区别在于，由 `define()` 函数定义的常量是在运行时创建的。相反，使用 `const` 语句创建的常量是在编译时创建的。这种情况会产生一些后果。例如，使用 `const` 语句创建常量的命令不能放在条件语句中，但使用 `define()` 函数则可以。另一方面，`define()` 函数不能在类和接口（将在本书后续章节中讨论）中用于创建常量字段；在这种情况下，需要使用基于 `const` 语句的表达式。

## 静态变量

*静态变量*的概念相当有趣。这种变量使用 `static` 关键字声明。函数终止后，它不会从内存中移除，并且在每次后续函数调用中，都使用同一个变量。程序清单 6-6 中的程序展示了如何使用静态变量。

```
",calc(),"\n";
}
?>

静态变量
```

程序执行的结果可能看起来有点出乎意料。

```
程序输出（来自程序清单 6-6）
[1] calc() -> 100
[2] calc() -> 200
[3] calc() -> 300
[4] calc() -> 400
[5] calc() -> 500
```

简单来说，一切都很明了。你有一个 `calc()` 函数，调用时不带任何参数。第一次调用该函数时，结果是 `100`。第二次调用时，结果是 `200`。第三次调用时，函数返回 `300`，依此类推。因此，你使用相同的指令，但每次都得到一个新的（可预测的）结果。为了理解为什么会这样，我们来分析一下 `calc()` 函数的代码。

在函数体中，`static $value=0` 命令创建了一个值为 `0` 的静态变量 `$value`。然后，通过 `$value+=100` 命令，该值增加 `100`，并作为函数的结果返回。

来看看第一次调用函数时发生了什么。很简单：`$value` 变量被创建，之后它获得值 `100`，然后函数返回结果。至此，函数的执行完成。如果 `$value` 变量是普通的（非静态的），那么它会被从内存中移除，下次调用函数时，一切都会从创建新的 `$value` 变量开始。但该变量是静态的。因此，函数终止后，该变量不会从内存中删除。

而且，下次调用 `calc()` 函数时，使用的就是这个变量，并且其值是上一次函数调用结束时所拥有的值。该值增加 `100` 后，作为结果返回。结果是，每次调用 `calc()` 函数时，`$value` 变量的值都会增加 `100`。因此，每次函数都会返回一个新的值。

| 详情 |
| --- |
| 用于将静态变量 `$value` 初始化为零值的命令仅执行一次，即第一次调用函数时。另外值得一提的是，`$value` 变量（尽管是静态的）仅在函数体内部可用。 |

## 使用文件

使用 PHP 的另一个重要方面与文件操作有关。我们来讨论一下从文件获取数据和将数据写入文件的最常用技术。

首先，需要打开文件。为此，可以使用 `fopen()` 函数。该函数的第一个参数是指定文件路径的字符串。第二个参数是包含特殊字符的字符串。该参数决定了使用文件的访问模式。表 6-1 列出并描述了 `fopen()` 函数第二个参数的组合。

**表 6-1** 使用文件的访问模式

| 指令 | 访问模式 |
| --- | --- |
| `'r'` | 以只读方式打开文件。读取指针设置在文件开头。 |
| `'r+'` | 以读写方式打开文件。读/写指针设置在文件开头。 |
| `'w'` | 以写入方式打开文件。写入指针设置在文件开头。文件原有内容将被删除。如果文件不存在，则会创建它。 |
| `'w+'` | 以读写方式打开文件。读/写指针设置在文件开头。文件原有内容将被删除。如果文件不存在，则会创建它。 |
| `'a'` | 仅以写入方式打开文件。写入指针设置在文件末尾。如果文件不存在，则会创建它。 |
| `'a+'` | 以读写方式打开文件。读/写指针设置在文件末尾。如果文件不存在，则会创建它。 |

| 详细信息 |
| --- |

## 总结

-   引用提供了对变量的替代访问方式。在创建对变量的引用时，变量名前加上 `&` 指令被赋值给该引用。这种引用称为硬引用。
-   使用软引用时，某个变量会被赋予一个包含另一个变量名称（不带 `$` 符号）的字符串。在第一个变量名称前加上第二个 `$` 符号，可以访问其名称包含赋给第一个变量的字符串的那个变量。
-   除了变量，还可以使用常量。常量在首次赋值后其值不能更改。常量可以使用 `define()` 函数或 `const` 语句创建。常量名称中不使用 `$` 符号。
-   函数可以使用全局变量。这些变量在函数作用域之外也是可访问的。在函数体内，使用 `global` 语句声明全局变量。
-   函数可以使用静态变量。（使用了 `static` 关键字来声明此类变量。）静态变量在函数首次调用时被创建并获得其初始值。函数终止后，静态变量不会从内存中移除，下次调用该函数时，该变量会被再次使用。
-   在某些情况下，会使用多行字符串。在这种情况下，变量被赋值的表达式以 `<<<` 指令开头，后跟某个标签。文本在多行中输入，无需引号。表达式以相同的标签和分号结束。这相当于双引号字符串。如果标签首次出现时被括在单引号中，则得到单引号字符串的类似形式。
-   有一组函数用于处理文件。`fopen()` 函数用于打开文件。第一个参数指定文件的完整路径，第二个参数指定访问模式。文件处理完毕后，使用 `fclose()` 函数关闭它。
-   要导入文件的内容，需要在程序中添加 `require_once` 指令，后面跟上包含导入文件名称的字符串。
