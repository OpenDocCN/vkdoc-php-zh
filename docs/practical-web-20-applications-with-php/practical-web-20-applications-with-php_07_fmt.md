# 注意

要使此 `VirtualHost` 指令生效，在加载此虚拟主机之前，您必须已在主 Web 服务器配置中包含 `NameVirtualHost 192.168.0.80`。

您可能还想向配置中添加其他指令，但这已经是一个相当标准的配置。它允许您根据需要通过 `.htaccess` 文件（由于 `AllowOverride` 指令）按目录覆盖配置，并告知 PHP 模块在何处查找包含的文件。在此示例中，它将首先在当前目录中查找，然后在 `/var/www/phpweb20/include` 目录中查找，最后在 PEAR 库中查找。请注意，PEAR 的具体位置可能因您的 Linux 发行版或操作系统而异。
