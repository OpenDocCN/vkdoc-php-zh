## 循环

PHP 中有四种循环结构。它们用于多次执行特定的代码块。与条件判断中的 `if` 语句一样，如果代码块中只有一条语句，则循环的花括号可以省略。

## While 循环

`while` 循环仅在条件为 true 时执行代码块，并且只要条件保持为 true 就会继续循环。请注意，条件仅在每次迭代（循环）开始时检查。

```
$i = 0;
while ($i < 10) { echo $i++; } // 0-9
```

## Do-while 循环

`do-while` 循环的工作方式与 `while` 循环相同，区别在于它在代码块执行之后检查条件。因此，它至少会执行一次代码块。请记住，此循环以分号结尾。

```
$i = 0;
do { echo $i++; } while ($i < 10); // 0-9
```

## For 循环

`for` 循环用于按特定次数执行代码块。它使用三个参数。第一个参数初始化一个计数器，并在循环开始前始终执行一次。第二个参数包含循环的条件，在每次迭代前检查。第三个参数包含计数器的增量，在每次迭代结束时执行。

```
for ($i = 0; $i < 10; $i++) { echo $i; } // 0-9
```

`for` 循环有多种变体，因为其中任何一个参数都可以省略。例如，如果省略第一个和第三个参数，它的行为与 `while` 循环相同。

```
for (;$i < 10;) { echo $i++; }
```

第一个和第三个参数也可以使用逗号运算符（`,`）拆分为多个语句。

```
for ($i = 0, $x = 9; $i < 10; $i++, $x--) {
  echo $x; // 9-0
}
```

`sizeof` 函数用于获取数组中的元素数量。结合 `for` 循环，它可以用来遍历数字索引数组。

```
$a = array(1,2,3);

for($i = 0; $i < sizeof($a); $i++) {
  echo $a[$i]; // "123"
}
```

如果不需要跟踪迭代次数，`foreach` 循环提供了更简洁的语法。遍历关联数组时也必须使用此循环。

## Foreach 循环

`foreach` 循环提供了一种遍历数组的简便方法。在每次迭代中，数组中的下一个元素被赋值给指定的变量（迭代器），并且循环会持续执行，直到遍历完整个数组。

```
$a = array (1,2,3);

foreach ($a as $v) {
  echo $v; // "123"
}
```

`foreach` 循环有一个扩展用法，通过添加一个键变量，后跟双箭头运算符（`=>`），再放到迭代器前面，来同时获取键名或索引。

```
$a = array ('one' => 1, 'two' => 2, 'three' => 3);

foreach ($a as $k => $v) {
  echo "$k => $v <br>";
}
```

## 替代语法

与条件语句一样，循环中的花括号可以重写为替代语法，使用冒号和 `endwhile`、`endfor` 或 `endforeach` 关键字之一。

```
while ($i < 10): echo $i++; endwhile;

for ($i = 0; $i < 10; $i++): echo $i; endfor;

foreach ($a as $v): echo $v; endforeach;
```

这样做的主要好处是提高了可读性，尤其是在处理较长的循环时。

## Break

有两个特殊关键字可以在循环内部使用——`break` 和 `continue`。`break` 关键字用于结束循环结构的执行。

```
for (;;) { break; } // end for
```

它可以接受一个数值参数，用于指定要跳出多少层嵌套的循环结构。

```
$i = 0;
while ($i++ < 10)
{
  for (;;) { break 2; } // end for and while
}
```

## Continue

`continue` 关键字可用于任何循环语句中，以跳过当前循环的剩余部分，并继续执行下一次迭代的开头。



```
while ($i++ < 10) { continue; } // 开始下一次迭代
```

该关键字可以接受一个参数，用于指定它应该跳过多少层外部循环的末尾。

```
$i = 0;
while ($i++ < 10)
{
  for (;;) { continue 2; } // 开始下一次 while 迭代
}
```

与许多其他语言不同，`continue` 语句同样适用于 `switch` 语句，其行为与 `break` 相同。因此，要从 `switch` 内部跳过一次迭代，需要使用 `continue 2`。

```
$i = 0;
while ($i++ < 10)
{
  switch ($i)
  {
    case 1: continue 2; // 开始下一次 while 迭代
  }
}
```

Goto

PHP 5.3 中引入的第三个跳转语句是 `goto`，它可以跳转到指定的标签。标签是一个名称后跟一个冒号（`:`）。

```
goto myLabel; // 跳转到标签处
myLabel:      // 标签声明
```

目标标签必须位于同一个脚本文件和作用域内。因此，`goto` 不能用于跳入循环结构中，只能跳出。

```
loop:
while (!$finished)
{
  // ...
  if ($try_again) goto loop; // 重新开始循环
}
```

通常，最好避免使用 `goto` 语句，因为它往往会使执行流程难以追踪。
