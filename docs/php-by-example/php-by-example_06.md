## 变量入门

在其“经典”意义上，变量是一块命名的内存区域，可以通过名称来访问它，以读取内存中的值或向其中写入新值。在 PHP 中，创建变量非常简单：你只需要给变量赋值即可。换句话说，当给变量赋值时，该变量就出现在程序中了。

PHP 中有一条规则：变量的名称以 `$` 符号开头（这个符号是变量名的一部分）。要为一个变量赋值，你需要指定变量名，然后加上赋值运算符（等号 `=` ），最后是该变量的值。例如，命令 `$number = 123` 将整数值 `123` 赋给变量 `$number`，命令 `$text = "Hello, PHP"` 将文本值 `"Hello, PHP"` 赋给变量 `$text`。

注释是程序中的文本，它是写给程序员看的，在程序执行时会被忽略。换句话说，注释用于解释命令的用途，但不会影响程序的执行。

除了注释之外，程序中还有几条值得注意的命令。第一条是 `$number = 123` 指令，它创建了值为 `123` 的变量 `$number`。此外，还有两条用于在终端中显示消息的命令。虽然两条命令都使用了 `print()` 函数，但它们有所不同。

![](img/613529_1_En_2_Figc_HTML.jpg) 信息图标。**注意**

这次，你在没有使用括号的情况下向 `print()` 函数传递了一个参数。

这两个命令的区别如下：在第一种情况下，要显示的文本被括在单引号中。在第二种情况下，文本被括在双引号中。原则上，在这两种情况下，你得到的是文本。这意味着单引号和双引号是创建文本的两种不同方式。它们几乎等同。但也有一些差异存在。在这个例子中，两个文本值（`'The variable $number: '` 和 `"the value $number"`）都包含了变量 `$number` 的名称。因此，如果变量名出现在用单引号括起来的文本中，那么在终端显示文本时，将显示变量名本身。如果变量名被放在用双引号括起来的文本中，那么在输出窗口中显示时，会将其值替换进去，而不是显示变量名。因此，在执行 `print 'The variable $number: '` 命令时，屏幕上显示的就是单引号中的文本。但执行 `print "the value $number"` 命令时，文本中的变量名 `$number` 会被其值 `123` 替换。

![](img/613529_1_En_2_Figd_HTML.jpg) 信息图标。**注意**

调用 `print()` 函数的结果是，该函数接收的参数的值会显示在输出窗口中。但输出区域中不会有换行符。因此，在讨论的这个程序中，两条消息会依次显示，位于同一行。

值得注意的是，PHP 中的许多常见任务可以通过不同方式完成。例如，有几种注释类型、几种创建文本字面量的方法、几种显示消息的指令，这个列表还不完整。这种情况在 PHP 中很典型。从这个意义上说，PHP 语言非常便利和“舒适”，如果这样的定义适用于一门编程语言的话。

| --- |
| 除了变量，你还可以使用常量。常量与变量的不同之处在于，常量的值不能更改。要创建一个常量，请使用 `define()` 函数，该函数的第一个参数是常量的名称，第二个参数是常量的值。常量名称中不使用美元符号 `$`。常量名称通常由大写字母组成。例如，命令 `define("NUMBER", 123)` 定义了值为 `123` 的常量 `NUMBER`。你可以使用 `defined()` 函数检查具有特定名称的常量是否存在。该函数的参数是常量的名称（放在引号中）。 |



## 基本数据类型

如前所述，变量本身没有类型。然而，变量所引用的值是有类型的。你可以使用 `gettype()` 函数来确定变量所引用的值类型。变量作为参数传递给该函数。函数的返回值是一个包含类型名称的文本。具体来说，整数是 `integer` 类型，实数（浮点数）是 `double` 类型，文本是 `string` 类型。此外还有 `boolean` 类型（逻辑值 `true` 或 `false`）、`array`（数组）、`object`（对象）、`resource`（对外部资源的引用）以及 `NULL`（表示变量没有值）。以下是一个说明这种情况的小示例，如代码清单 2-2 所示。

```
<?php
$value=123;
echo $value," => type ",gettype($value),"\n";
// 一个文本:
$value="text";
echo $value," => type ",gettype($value),"\n";
// 一个实数:
$value=12.3;
echo $value," => type ",gettype($value),"\n";
// 空引用:
echo $VALUE," => type ",gettype($VALUE),"\n";
?>
代码清单 2-2
值的类型
```

程序执行结果如下。

```
程序的输出（来自代码清单 2-2）
123 => type integer
text => type string
12.3 => type double
=> type NULL
```

当使用 PHP 8 及更高版本时，关于未知变量 `$VALUE` 会有一条警告信息，该变量出现在 `type` 和 `NULL` 标记之间。

这里你面对的是一个简单程序。`$value` 变量先后被赋以不同的值，并且为每个值确定其类型。要确定类型，请使用 `gettype()` 函数，其参数就是变量 `$value`。

![](img/613529_1_En_2_Fige_HTML.jpg)一个信息图标。注意

最后一条命令值得注意。它使用了 `$VALUE` 变量。这与变量 `$value` 完全不同，因为变量名是区分大小写的（变量名中使用大写字母还是小写字母很重要）。因此，`$value` 和 `$VALUE` 是不同的变量。

为了显示值，使用了基于 `echo` 关键字的指令。在该指令右侧，列出了用逗号分隔的值，它们将在输出区域中被显示。具体来说，这些值分别是 `$value` 变量的当前值、字符串 `" => type "`、由 `gettype($value)` 语句计算得出的 `$value` 变量值类型的名称，以及转义序列 `"\n"`。

| 详细信息 |
| --- |
| `\n` 指令是一种所谓的转义序列。如果这样的指令被放在一个用双引号括起来的文本中（这很重要！），那么在显示文本时，会在相应位置执行换行。因此，应该“打印”文本 `"\n"` 来换行。`\n` 转义序列并非 PHP 中唯一的转义序列。例如，`\t` 指令会在文本中插入一个制表符。 |

最后一条语句 `echo $VALUE," => type ",gettype($VALUE),"\n"` 需要特别解释。如前所述，PHP 将变量名中的大写字母和小写字母解释为不同的字符（这与函数名不同）。因此，`$VALUE` 变量与 `$value` 变量毫无关系。

在前面的语句中，试图在输出区域显示 `$VALUE` 变量的值，尽管该变量尚未被赋值。该变量没有值；换句话说，该变量的值是 `NULL` 引用。这证实了以 `$VALUE` 作为参数调用 `gettype()` 函数的结果。当试图显示 `$VALUE` 变量的值时，由于它不存在，输出区域中不会显示任何内容。从这个意义上说，PHP 语言对于变量没有值的情况是相当“宽松”的。

![](img/613529_1_En_2_Figf_HTML.jpg)一个信息图标。注意

有不少函数是专门用来操作数据类型的。例如，`settype()` 函数用于设置类型。`get_class()` 函数允许你确定对象的类。你可以使用 `function_exists()` 函数检查某个函数是否存在。使用 `method_exists()` 函数可以检查某个方法是否存在。你可以使用 `is_callable()` 函数检查一个变量是否引用了可调用对象。还有一组函数允许你检查数据是否属于特定类型：`is_array()`（变量是否引用数组）、`is_bool()`（变量是否引用布尔值）、`is_float()`（变量是否引用实数）、`is_int()`（变量是否引用整数）、`is_null()`（变量的值是否为空引用）、`is_numeric()`（变量是否引用数字或包含数字文本表示的字符串）、`is_object()`（变量是否引用对象）、`is_resource()`（变量是否引用资源）、`is_scalar()`（变量是否引用标量值）、`is_string()`（变量是否引用文本）。

| 详细信息 |
| --- |
| 除了使用 `settype()` 函数显式改变值的类型之外，你还可以使用 `(type_name)` 指令（括号中的类型名称），将其放在要转换类型的值之前。例如，`(int)` 或 `(integer)` 指令用于强制转换为整数类型，`(bool)` 和 `(boolean)` 指令用于强制转换为布尔类型，`(string)` 指令用于强制转换为文本类型，而将值强制转换为实数类型可以使用 `(float)`、`(double)` 或 `(real)` 指令（最后一条指令在 PHP 8 中已被弃用）。此外，还有其他选项。 |



## 算术运算

可以对不同类型变量和值执行一些基本运算。传统上，所有运算可分为算术运算、逻辑运算、比较运算和位运算。让我们也探讨文本运算、赋值运算和三目运算符。

| 详细信息 |
| --- |
| 有时，你需要检查某个变量是否被赋值。在这种情况下，可以使用`isset()`函数（待检查的变量名作为函数参数传入）。要删除变量，使用`unset()`函数。 |

值得一提的是，算术运算符`+`、`-`、`*`和`/`分别用于计算两个数的和、差、积和商。使用`**`运算符（或`pow()`函数）进行幂运算。`%`运算符计算除法的余数。

除了这些二元运算符，还可以使用一元递增`++`和递减`--`运算符。此外，它们有前缀和后缀两种形式。

| 详细信息 |
| --- |
| 二元运算符有两个操作数。一元运算符有一个操作数。递增和递减运算符的前缀形式和后缀形式的区别在于运算符是放在操作数之前（前缀形式）还是放在操作数之后（后缀形式）。 |

关于对操作数的“作用”，后缀形式和前缀形式是等价的。递增运算符`++`将操作数的值增加 1，递减运算符`--`将操作数的值减少 1。例如，如果`$number`变量的值是`100`，则执行`$number++`语句后，该变量的值将为`101`。如果使用`++$number`代替`$number++`语句，结果也是一样的：`$number`变量获得新值`101`。然而，后缀形式和前缀形式之间存在差异。使用递增/递减运算符后缀形式的表达式的值是操作数的“旧”值（应用运算符之前的值）。使用递增/递减运算符前缀形式的表达式的值是操作数的“新”值（应用运算符后计算出的值）。

| 详细信息 |
| --- |
| 假设存在命令`$A=100`和`$B=100`。执行`$a=$A++`和`$b=++$B`命令后，`$a`的值为`100`，而`$b`、`$A`和`$B`的值均为`101`。这是因为在执行`$A++`和`++$B`语句后，`$A`和`$B`变量都增加了 1（并得到值`101`），但`$A++`表达式的值为`100`，而`++$B`表达式的值为`101`。因此，`$a`变量的值为`100`，`$b`变量的值为`101`。 |

## 比较运算

除了算术运算，还有比较运算。这些运算通过以下运算符实现：`<`（*小于*）、`<=`（*小于或等于*）、`>`（*大于*）、`>=`（*大于或等于*）、`==`（*等于*）、`!=`（*不等于*）、`===`（*全等于*）、`!==`（*不全等于*）以及`<=>`比较运算符。列表中的大多数运算符都是传统的，易于理解。可能例外的是`<=>`运算符、*等于/不等于*运算符以及*全等于/不全等于*运算符。

需要注意的是，当使用运算符（包括比较运算符）处理不同类型的操作数时，会自动进行类型转换。这意味着为了执行语句，会应用某些算法来自动转换语句中操作数的值。使用`==`和`!=`运算符时，比较涉及自动类型转换。例如，如果使用`==`运算符比较布尔值`true`和整数值`1`，会得到`true`。原因是非零值被解释为`true`，零值被解释为`false`。因此，不仅`true==1`语句返回`true`，而且`true==2`、`true==-10`和`false==0`也都返回`true`。

| 详细信息 |
| --- |
| 如果尝试打印布尔值`true`，会打印出值`1`。此外，布尔值`false`根本不显示。取而代之的是打印空文本（即不含字符的文本）。 |

但还有更奇特的情况。例如，`123=="123"`语句中比较数字`123`和文本`"123"`是否相等，返回`true`。这是因为这里涉及自动将数字的文本表示`"123"`转换为整数值`123`。

所描述的比较模式可能方便也可能不方便，这取决于要解决问题的上下文。如果希望比较不涉及自动类型转换，则应使用`===`和`!==`运算符。例如，`true===1`、`false===0`和`123==="123"`表达式都返回`false`。

![](img/613529_1_En_2_Figq_HTML.gif)一个铃铛图标。**PHP 8 标准**

在 PHP 8 中，值的比较规则有所改变。主要创新是空文本值不再被解释为等于零。例如，在 PHP 8 中，`0==""`表达式的值为`false`，而在 PHP 的早期版本中，`0==""`表达式的值为`true`。

关于比较运算`<=>`，其执行方式如下。假设有一个类似`$A<=>$B`的表达式。如果`$A`的值小于`$B`的值，则此表达式的结果为`-1`。如果`$A`变量的值大于`$B`变量的值，则`$A<=>$B`表达式的计算结果为`1`。最后，如果`$A`和`$B`变量的值相同，则`$A<=>$B`表达式的值为`0`。

| 详细信息 |
| --- |
| `$A<=>$B`形式的运算可以理解为返回`$A`和`$B`值之差的符号。此外，请记住该运算涉及自动类型转换。因此，例如，`"123"<=>123`表达式的结果为`0`，因为文本`"123"`被自动转换为数字`123`。相应地，`"123"<=>12`的值为`1`。`true<=>123`表达式的值为`0`，因为非零整数值`123`被自动转换为布尔值`true`。 |



## 逻辑运算

逻辑运算符旨在对布尔类型的值执行运算（这里，你还必须牢记自动类型转换）。要执行逻辑运算，需要使用二元运算符 `&&`（*逻辑与*）和 `||`（*逻辑或*），以及一元运算符 *逻辑非* `!`。

形式为 `$A&&$B` 的表达式的值为 `true`，前提是操作数 `$A` 和 `$B`（考虑自动类型转换后）均为 `true`。如果至少有一个操作数为 `false`（或在自动类型转换后被解释为 `false` 值），那么 `$A&&$B` 表达式的结果为 `false`。

> **注意：** 信息图标。  
> 简单来说，表达式 `$A&&$B` 仅在两个操作数 `$A` 和 `$B` 都为 `true` 时才为 `true`。

形式为 `$A||$B` 的表达式的值为 `true`，前提是操作数 `$A` 或 `$B` 中至少有一个为 `true`，或者其值由于自动类型转换被转换为 `true`。如果两个操作数都为 `false`，则整个表达式的值为 `false`。

> **注意：** 由此可见，只要表达式中的操作数至少有一个为 `true`，`$A||$B` 表达式的值即为 `true`。

值得一提的是，基于运算符 `&&` 和 `||` 的表达式是按照简化方案从左到右计算的。也就是说，如果在计算第一个操作数后就能确定整个表达式的结果，那么第二个操作数将不再计算。

| 详细信息 |
| --- |
| 如果在形如 `$A&&$B` 的表达式中，第一个操作数 `$A` 为 `false`，那么无论第二个操作数 `$B` 的值如何，整个表达式的值都为 `false`。在形如 `$A||$B` 的表达式中，如果第一个操作数 `$A` 为 `true`，那么无论第二个操作数 `$B` 的值如何，整个表达式的值都为 `true`。因此，在这些情况下，不会计算第二个操作数的值。 |

逻辑非运算符 `!` 是一元运算符。形如 `!$A` 的表达式的值为 `true`，前提是操作数 `$A` 的值为 `false`（或由于自动类型转换被转换为 `false`）。如果操作数 `$A` 的值为 `true`（或被归为该值），那么 `!$A` 表达式的结果为 `false`。

| 详细信息 |
| --- |
| 运算符 `&&` 和 `||` 有其“孪生兄弟”，分别是 `and` 和 `or` 运算符。这些运算符的优先级低于 `&&` 和 `||` 运算符。在某些情况下，使用它们可能更方便。 |

## 位运算

位运算是在数字的二进制（按位）表示级别上执行的。因此，要理解执行此类运算的后果，至少需要对数字如何在二进制系统中编码有基本的了解。

| 详细信息 |
| --- |
| 在二进制表示中，数字被写成一串由零和一组成的序列。形式上，可以表示为 `a_n a_{n-1} ... a_2 a_1 a_0`，其中参数 `a[0]`、`a[1]`、……、`a[n]` 的取值可以是 0 或 1。要将正数从二进制转换为十进制，可以使用公式 `a_n a_{n-1} ... a_2 a_1 a_0 = a_0 * 2⁰ + a_1 * 2¹ + a_2 * 2² + ... + a_{n-1} * 2^{n-1} + a_n * 2^n`。 |

让我们来看看基本的位运算符，从*按位与*运算符 `&` 开始。假设要计算像 `$A&$B` 这样的表达式，其结果是一个数字。该数字的二进制码按照以下规则计算：比较操作数 `$A` 和 `$B` 的二进制码，具体来说，比较这些二进制码中对应的位。比较结果为表达式值的数字生成一个位。

| 详细信息 |
| --- |
| 计算机使用固定数量的位来存储数字。因此，无论值是什么，它们都是由相同数量的位来实现的，并且这些位之间存在一一对应关系。 |

使用 `&` 运算符时，如果被比较的两个位都是 `1`，则得到位 `1`。如果至少有一个位是 `0`，则结果位为 `0`。例如，表达式 `13&21` 的值是数字 `5`。解释如下：数字 `21` 的二进制码是 `10101`。数字 `13` 的二进制码是 `01101`。表达式 `13&21` 的结果是这样计算的：将码 `01101` 和 `10101` 逐位比较，如果两个位都是 `1`，则输出 `1`。在所有其他情况下（如果至少有一个位是 `1`），则输出 `0`。

```

```

最终得到码 `00101`，它对应于数字 `5`。

> **注意：** 数字的二进制表示仅使用了五个位置（位）。所有其他（更高位的）位都假定为零。

使用*按位或*运算符 `|` 比较数字中的位时，如果至少有一个位是 `1`，则得到 `1`。换句话说，像 `$A|$B` 这样的表达式的结果是一个数字，其二进制码是通过比较数字 `$A` 和 `$B` 的二进制码得到的。在得到的数字中，如果 `$A` 和 `$B` 中相同位置的被比较位至少有一个是 `1`，则该位置的位为 `1`。如果两个被比较位都是 `0`，则结果位为 `0`。因此，表达式 `13|21` 的结果是 `29`（二进制码 `11101`）。

```
| 01101
```

*按位异或*运算符 `^` 与*按位或* `|` 的不同之处在于：当且仅当被比较的两个位中有一个是 `1` 时，输出才为 `1`。在所有其他情况下（两个位都是 `1` 或两个位都是 `0`），结果位为 `0`。例如，表达式 `13²¹` 的结果计算如下。

```
^ 01101
```

码 `11000` 对应于 `24`，这就是表达式 `13²¹` 的值。

*按位取反*运算符 `~` 是一元运算符。形如 `~$A` 的表达式的结果是一个数字，其二进制码是通过将数字 `$A` 的码中的零替换为一、一替换为零而得到的。在这种情况下，数字 `$A` 的码本身不变，而是计算出新的值。



稍后你将看到，`~$A` 表达式的值是数字 `-($A+1)`。例如，`~5` 表达式的值是数字 `-6`。

*按位移位运算*（将数字的二进制表示向右或向左移动若干位）通过 `<<`（*左移*）和 `>>`（*右移*）运算符执行。`$A<<$n` 形式表达式的值是一个数字，其二进制码是将数字 `$A` 的二进制码向左移动 `$n` 位得到的。左侧的位丢失，右侧的位用 0 填充。`$A>>$n` 形式表达式的值是一个数字，其二进制码是将数字 `$A` 的二进制码向右移动 `$n` 位得到的。右侧的位丢失，左侧的位用符号位的值填充（第一位，正数为 `0`，负数为 `1`）。

| 详情 |

| 负数的编码有其特殊性。为了理解编码原理，假设数字用 8 位存储。假设你有一个代码。例如，令代码为 `00010011`（数字 `19`）。对该代码执行按位取反（将 `1` 替换为 `0`，将 `0` 替换为 `1`）。你会得到代码 `11101100`。现在，将这两个代码相加。二进制系统中的加法规则很简单：`0+0=0`，`1+0=1`，`0+1=1`，`1+1=10`。最后一项相当于 `2+2=4`，但只是用二进制代码书写。因此，如果将代码 `00010011` 和 `11101100` 相加，你会得到由 1 组成的代码 `11111111`。接下来，向该代码加一（代码 `00000001`）。你会得到 `100000000`。但这里，你必须记住你是在处理一台计算机，假设它使用 8 位存储数字。然而，代码 `100000000` 中有九位数字。因此，数字的开头根本没有位来记住这个 `1`，它就丢失了。你会得到由 0 组成的代码 `00000000`。这个数字是 `0`。如果采用不同的源代码和不同的位数来存储数字，结果将是相同的。你得到了什么？如果你取某个正数 `$X`，对其代码取反（`~$X` 表达式），将这些代码相加，然后再加 `1`，你最终会得到 `0`。换句话说，无论 `$X` 的值是多少，都存在关系 `$X+(~$X)+1=0`。很容易看出，表达式 `(~$X)+1` 可以等同于数字 `-$X` 的代码（与 `$X` 相反的数字）。因此，如果你需要得到一个负数的二进制代码，你应该取相反正数的代码，对该代码取反，然后加 `1`。数字二进制代码的第一位用于确定符号：对于正数，该位为 `0`；对于负数，该位为 `1`。对于正数，二进制表示根据标准数学规则转换为十进制值。如果你需要将负数的二进制代码转换为十进制表示，则应遵循以下步骤。首先，对代码取反。然后加 `1`。得到的值根据正数的转换规则转换为十进制表示。最后一步，在计算出的数字前加上负号。 |

## 字符串操作

处理文本是一个广泛的主题。在此，我们只关注文本的基本操作以及你在不久的将来会用到的文本特性。

> **注意：** 信息图标。  
> 你已经知道文本字面量可以用单引号或双引号括起来。在后一种情况下，如果文本包含变量名，其值会替换变量名。你还可以在此类文本中插入转义序列，它们由反斜杠 `\` 和定义某种操作的字符组成。例如，`\n` 是换行指令，`\t` 是制表符。你可以使用 `\\`、`\$`、`\'` 和 `\"` 分别插入反斜杠 `\`、美元符号 `$`、单引号 `'` 和双引号 `"`。

经常对文本执行的基本操作之一是字符串连接。在 PHP 中，字符串连接可以通过使用 `.` 运算符（“点”）来完成。例如，表达式 `"Learning"`.`" PHP"` 的结果是字符串 `"Learning PHP"`。同样的操作也可以使用变量完成：形式为 `$A.$B` 的表达式的值，是将 `$A` 和 `$B` 变量的文本值合并计算出的文本。

| 详情 |
| --- |
| 如果 `$A` 和 `$B` 变量具有非文本值，则 `$A.$B` 语句会尝试将这些值转换为文本类型，然后再进行连接。另外值得一提的是，与许多其他编程语言不同，PHP 不使用 `+` 运算符来连接文本值。此外，对此运算符应用连接文本值可能会导致错误或意外结果。例如，表达式 `"1"+"2"` 不会像你期望的那样产生文本 `"12"`，而是产生数值 `3`。原因是 `+` 运算符是一个算术运算符，因此操作数 `"1"` 和 `"2"` 会被自动转换为整数 `1` 和 `2`。之后，这些数字相加。因此，要获得文本 `"12"`，你应该使用 `"1"."2"` 语句。 |

另一个简单的字符串操作是确定字符串的长度。在这种情况下，使用 `strlen()` 函数。你可以使用 `==` 和 `!=` 运算符来比较字符串是否相等/不相等。但在这里，你应该记住自动类型转换。因此，使用 `===` 和 `!==` 运算符有时更有意义。

> **注意：** 信息图标。  
> 你也可以使用 `strcmp()`（区分大小写的比较）和 `strcasecmp()`（不区分大小写的比较）函数来比较字符串。

还有几个其他重要的文本操作。其中，最主要的是从字符串中提取子字符串。为此，请使用 `substr()` 函数。该函数的参数是：一个字符串（从中提取子字符串）、子字符串起始字符的索引，以及（如果需要）子字符串的长度。如果你没有指定最后一个参数，子字符串将被提取到字符串的末尾。

`str_replace()` 函数用于在字符串中替换子字符串。该函数将第三个参数指定的字符串中，由第一个参数指定的子字符串替换为第二个参数指定的子字符串。

> **注意：** 信息图标。  
> 你可以通过在文本值后面的方括号中指定字符索引来访问字符串中的单个字符。例如，如果 `$A` 变量有一个文本值，那么表达式 `$A[$n]` 是对索引为 `$n` 的字符的引用。字符索引从零开始。

## 赋值

在 PHP 中，赋值运算符会返回一个值。这意味着带有赋值运算符的表达式可以是另一个表达式中的操作数。特别是，形式为 `variable=value` 的表达式的值是表达式左侧变量的值。例如，执行 `$A=($B=100)+1` 指令后，`$A` 的值为 `101`，`$B` 的值为 `100`。

| 详情 |
| --- |
| `$A=($B=100)+1` 命令将表达式 `($B=100)+1` 赋值给变量 `$A`。这是两项之和：`($B=100)` 和 `1`。当计算表达式 `$B=100` 时，变量 `$B` 获得值 `100`，而表达式 `($B=100)` 也具有相同的值。因此，变量 `$A` 被赋值为 `100` 和 `1` 的和，即数字 `101`。 |

还存在简化形式的赋值运算符。它们允许你简化执行形式为 `$A=$A` ![](img/613529_1_En_2_Figr_HTML.gif) `$B` 的赋值语句的过程，其中 ![](img/613529_1_En_2_Figs_HTML.gif) 是二元算术或位运算符，或是字符串连接运算符。因此，你可以使用 `$A` ![](img/613529_1_En_2_Figu_HTML.gif)`= $B` 指令来代替诸如 `$A=$A` ![](img/613529_1_En_2_Figt_HTML.gif) `$B` 之类的语句。例如，你可以使用 `$A+=$B` 命令代替 `$A=$A+$B` 语句，而 `$A.=$B` 命令是 `$A=$A.$B` 指令的等价形式。



## 三元运算符

运算符通常是一元（只有一个操作数）或二元（有两个操作数）。但有一种运算符拥有三个操作数，被称为*三元*运算符。通常来说，如果你不纠结于术语，可以将其视为一种特殊的语法结构，它允许你根据一个真或假的条件来为变量赋值。三元运算符的语法如下。

```
condition?value_1:value_2
```

首先，放置条件（通常是一个返回布尔值的表达式）。然后输入一个问号（`?`）、条件为真时返回的值、一个冒号（`:`），以及条件为假时返回的值。上述结构可以赋值给一个变量。例如，执行`$number=($x<0)?-10:10`语句后，如果`$x`的值小于零（即条件`$x<0`为真），则`$number`被设为`-10`，否则（条件`$x<0`为假）设为`10`。清单 2-4 展示了一个使用三元运算符的简单示例。

```
清单 2-4
三元运算符
```

这个程序很简单，但包含一些值得注意的指令。首先，`$number=(int)trim(fgets(STDIN))`语句从键盘读取用户输入的整数值，并将其写入`$number`变量。接着，利用三元运算符判断`$number`变量是偶数还是奇数，根据判断结果，为`$word`变量赋予`"even"`或`"odd"`的值。判断结果显示在屏幕上。

运行程序时，系统会提示你输入一个数字。程序等待用户输入。用户输入数字并按下回车键。之后，会显示另一条消息，其中包含该数字是偶数还是奇数的信息。根据用户输入的值（以粗体显示），程序执行的结果可能如下（输入奇数时）。

```

这是一个奇数
```

或者结果可能如下（输入偶数时）。

```
程序输出（来自清单 2-4）
输入一个数字：124
这是一个偶数
```

现在，我们来分析主要语句。首先，考虑`$number=(int)trim(fgets(STDIN))`指令。它基于对`fgets()`函数的调用，该函数从输入流中读取字符串。函数参数是标准输入流的常量标识符`STDIN`（在本例中，与控制台关联）。但`fgets()`函数会将任何输入值作为文本读取，而且文本可能包含特殊转义序列（如换行指令）。要删除所有“多余”字符，请使用`trim()`函数。它会移除作为参数传递的字符串中的所有前导和尾随空格及特殊字符。因此，如果用户输入一个整数，那么`trim(fgets(STDIN))`表达式包含该数字的文本表示形式。例如，如果用户输入数字`123`，则读取的值（考虑到对`trim()`函数的调用）是`"123"`。坦率地说，考虑到自动类型转换，即使使用该值也可以进行计算。

不过，为了可靠性，你需要将读取的文本转换为数字。为此，请在整个表达式前放置`(int)`指令。除了前面提到的`settype()`函数，这是另一种显式转换类型的方法。

| 详细信息 |
| --- |
| 除了使用从控制台为`$number`变量读取值的`$number=(int)trim(fgets(STDIN))`语句外，你也可以使用`$number=123`或`$number=124`语句，直接为`$number`变量显式赋值。 |

`$word=($number%2==0)?"even":"odd"`命令将`($number%2==0)?"even":"odd"`表达式（基于三元运算符）赋值给`$word`变量。如果`$number%2==0`条件为真（意味着`$number`变量除以`2`的余数为零），则`$word`变量获取文本`"even"`。如果`$number%2==0`条件为假，则`$word`变量获取文本`"odd"`。`$word`变量的值用于显示判断结果。

## 总结

*   变量是一个命名的内存块，允许你通过名称引用该内存块。要在 PHP 中创建变量，只需为变量赋值即可。变量名以`$`字符开头。
*   变量没有类型，但变量引用的值有类型。主要类型包括`integer`（整数）、`double`（实数）、`string`（文本）、`boolean`（布尔值）、`array`（数组）和`object`（对象）。你可以使用`gettype()`函数获取变量引用的值的类型。
*   对于显式类型转换，你可以使用`settype()`函数或在待转换的值前用括号指定最终类型。PHP 也具备自动类型转换系统。
*   基本运算使用算术、逻辑、位运算和比较运算符。*点*运算符用于连接字符串。
*   赋值运算符返回的值与变量（被赋值的变量）的值相同。存在简化形式的赋值运算符。
*   三元运算符返回一个值，该值取决于一个可以为真或假的具体条件。
*   `fgets()`函数可以从控制台读取数据。
