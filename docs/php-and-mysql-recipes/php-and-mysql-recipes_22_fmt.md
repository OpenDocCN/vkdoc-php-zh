## 引言

《PHP 5 食谱：问题-解决方案方法》于 2005 年发布。当时仍有许多网站使用较旧的 PHP 4 版本脚本语言，而 PHP 5 引入了许多新特性和一个完全重新设计的面向对象编程模型。自那以后，PHP 经历了多次版本发布，现在是时候更新内容与食谱，以涵盖自第一本书出版以来十一年间该语言新增的一些特性。PHP 的当前版本是 5.6.x，开发团队最近发布了 7.0 版本，与 PHP 5.6 相比，其性能有望翻倍，内存使用减半。从 PHP 5 直接跳到 PHP 7，跳过了主版本 6，是因为一个名为 PHP 6 的研究项目因复杂性和性能问题而被放弃。

PHP 是一种最常用于创建动态 Web 内容的脚本语言；事实上，大约 80% 的 Web 服务器都以某种形式安装了 PHP。PHP 也可用作 Shell 脚本的通用脚本语言，甚至可以通过 PHP-GTK 扩展创建桌面应用程序。本书将重点介绍 Web 和 Shell 脚本，面向初学者和高级用户。

PHP 是一种松散类型的脚本语言。在使用变量前无需声明，且变量类型会随内容变化而改变。作为一种脚本语言，PHP 在执行前无需编译或进行其他代码处理。当解释器被指示执行脚本时，它会加载文件内容、执行编译，并在未发现编译错误时执行代码。各种形式的缓存机制可以跳过编译阶段，直接进入执行流程。
