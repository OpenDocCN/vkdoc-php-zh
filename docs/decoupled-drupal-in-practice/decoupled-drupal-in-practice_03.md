# 致谢

撰写一本关于解耦式 Drupal 这样快速发展和成熟的主题的书籍，是一项充满挑战和回报、高潮和低谷的艰巨任务。我非常幸运在撰写这部作品时得到了许多个人和组织的支持。需要感谢的名字太多了，尽管致谢部分难免有不完整之处，但我将在此尽己所能。

首先，我要向 Acquia 以及我在首席技术官办公室（OCTO）的队友们表达我的感激之情，他们在这项工作中一直非常出色。我特别要感谢 Drupal 项目负责人兼 Acquia CTO Dries Buytaert，他在此项目过程中给予了坚定不移的支持。我还要感谢我亲爱的朋友 Wim Leers，他是很久以前我最早透露计划的人之一，他作为工程师的健康怀疑态度，以及作为朋友的道义支持，既调节了我的努力，也鼓舞了我的干劲。我还要感谢 Terrence Kevin O'Leary (tkoleary)，他是我在 Acquia 工作初期出色的导师，并且至今仍在推动我的所有职业追求。

我衷心感谢所有其他前 OCTO 和现任同事，他们在各个方面都非常出色：Alex Bronstein (effulgentsia)、Angie Byron (webchick)、ASH Heath、Chris Hamper (hampercm)、Emilie Nouveau (dyannenova)、Gabe Sullice (gabesullice)、Gábor Hojtsy、Jess (xjm)、Jesse Beach (jessebeach)、Lauri Eskola (lauriii)、Leah Magee、Mark Winberry (markwin)、Matt Grill (drpal)、Michael Meyers (michaelemeyers)、Moshe Weitzman (weitzman)、Sam Mortenson (samuel.mortenson)、Ted Bowman (tedbow) 和 Tim Plunkett (tim.plunkett)。

其次，我要感谢参与两项战略计划的所有贡献者，即使本书的墨迹尚未干透，他们也在规划 Drupal 的未来：API 优先计划和管理 UI 与 JavaScript 现代化计划。感谢上述未提及的 API 优先计划的协调员，没有他们，这本书是不可能的：Daniel Wehner (dawehner)、Mateu Aguiló Bosch (e0ipso) 和 Sebastian Siemssen (fubhy)。我还要感谢上述未提及的管理 UI 与 JavaScript 现代化计划的协调员：Cristina Chumillas (ckrina) 和 Sally Young (justafish)。

第三，我要感谢整个科罗拉多的 Drupal 社区，他们不懈地努力培育本地社区，激励了我过去 11 年参与 Drupal 并实现职业成长。我特别要向已故的 Rick Nashleanas 表达我的感激之情，他在科罗拉多斯普林斯的 Monarch Digital 雇佣了我，并迅速将我们整个 Web 业务转向 Drupal；还要感谢 Matt Tucker (ultimateboy)，他在十多年前首次向我介绍了 Drupal 的内部运作。我也要感谢我的高中英语老师 Jeff Flygare，他鼓励我追求 Web 开发和写作方面的兴趣。谁能想到我的这两个爱好会以这种方式结合在一起呢？

感谢 Apress 的编辑人员，他们从头到尾提供了出色的作者体验，特别是 Nancy Chen、Louise Corrigan 和 James Markham。也衷心感谢技术审校 Brandon Scott，他在整个过程中提供了可行且富有洞察力的反馈。

我还要深切感谢 Acquia 市场团队的成员，他们从一开始就支持我的图书项目，包括 Andrea Rosmarin、DC Denison 和 David Churbuck。我还向 Christopher Rogers、David Butler、Drew Robertson、Eric Williamson、Gigi Anderson、Lynne Capozzi、Molly Sloan、Reena Leone 和 Saša Zelenović致以诚挚的敬意和感谢。

我感谢分布在世界各地（纽约市、波士顿和全球各地）的所有朋友和家人，他们在过去一年中以无数种方式支持了我。你们的友谊和爱对我来说意味着一切。

最后但同样重要的是，我要向全球 Drupal 社区表达我无尽的感激之情，没有他们，支撑这本书的软件就不会存在。这个极其开放、热情和独特的开源社区在我心中激发了对自由和开源软件理念以及贯穿 Drupal 历史并至今仍在指导它的原则的不可估量的热情。

最后，我想对所有像我们这样在这个行业中被边缘化、代表不足且常常受到压迫的人说几句鼓励的话。我衷心希望这部作品能激励你为我们的讨论贡献自己的篇章，并介绍你宝贵的视角和独特的专长。我们迫切需要更多像你这样的声音，站在舞台中央，出现在会议舞台上，出现在我们当地的书店和图书馆里；我们必须继续拓宽空间，让像你这样的贡献得以蓬勃发展并激励他人。
