# 摘要

本章首先带您总体了解 PHP 框架，然后深入探讨了 CakePHP 及其主要特性。我详细介绍了该框架的核心约定，借助这些约定我们可以避免各种配置问题。在 Cake 的主要特性中，我重点强调了自动代码生成（或许还包括集成单元测试），因为这是本书的核心主题。
