# PHP 条件语句与变量处理  

条件语句的第一行使用两个等号检查`$title`的值。下一行使用单个等号将新值赋给`$title`。如果页面被调用的名称不是`index.php`，则花括号内的行将被忽略，`$title`保持其原始值。  

## 提示  
PHP 是区分大小写的，因此此解决方案仅在“index”全部为小写时有效。要进行不区分大小写的比较，请将前面代码的第四行修改为：`if (strtolower($title) == 'index') {`。函数`strtolower()`将字符串转换为小写——因此得名——并常用于进行不区分大小写的比较。转换为小写不是永久性的，因为`strtolower($title)`没有赋值给变量；它仅用于进行比较。要永久更改，需要将结果赋值回变量，就像最后一行中`ucwords($title)`被赋值回`$title`一样。要将字符串转换为大写，请使用`strtoupper()`。

## 注意事项  
绝大多数 PHP 网站托管在 Linux 服务器上，这些服务器将文件和目录（文件夹）名称视为区分大小写。然而，在 Windows 或 macOS 上本地开发时，文件和文件夹名称的处理方式不区分大小写。为避免在将文件部署到生产服务器时出现路径错误，我建议在命名文件和文件夹时仅使用小写。如果您想混合使用大小写，请确保拼写一致。
