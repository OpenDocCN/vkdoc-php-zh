# 致谢

撰写《PHP 解决方案》的初衷源于 Chris Mills，他是我在 Apress/friends of ED 共事多年的编辑，现任 Mozilla 公司高级技术文档撰写人，也是 Web 标准的热情倡导者。Chris 希望摒弃那种孤立解决方案的“菜谱式”编写模式，因为这种模式让读者对所涉及技术的实际用途知之甚少。本书能出版至第三版，足以证明当初这个想法是多么出色。谢谢你，Chris。

担任第二版和第三版编辑的 Ben Renow-Clarke 是 Chris 的继任者，他效仿 Chris，给予我充分的自由来按照自己的想法雕琢本书，但他总是能设身处地站在读者的角度，当某个解释不够清晰或某个章节结构混乱时，及时将我的思路引导到正确的方向。

我也非常感谢我的技术审阅者们：第一版的 Samuel Wright，第二版的 Kristian Besley 和 Jason Nadon，以及本版的 Paul Milbourne。每个版本都建立在前一版本的基础之上，因此前一版本审阅者的想法和建议在本版中得以延续。

若无 Apress 出版链中每一位成员的勤勉协助，要完成这样一本书是不可能的。我在此向他们所有人表示感谢。

最重要的是，我要感谢众多读者，是你们让本书取得了如此成功。欢迎加入俱乐部。我希望你们喜欢本书，并发现它在使用 PHP 构建动态网站时卓有成效。
