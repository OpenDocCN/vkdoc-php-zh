# 确保项目成功的基础

为了确保这个项目的成功，你需要想出一个聪明的方法来实现你在合同中签署的内容。你希望项目顺利且快速地开发，但最终目标是确保客户对你的工作感到满意。因此，你应该通过创建一个美观、功能强大且响应迅速的网站，为网站日益增多的访问者提供积极的网络体验。

需求很高，但对于当今的电子商务网站来说，这是正常的。为了最大化成功的机会，我们将尽可能多地分析和预见技术需求，并以一种能够以最小精力支持变更和扩展的方式实施解决方案。

本章为未来的`HatShop`网站奠定基础。我们将讨论你将使用的技术和工具，更重要的是，以及你将如何使用它们。

本章的目标是：

- 从技术角度分析项目
- 分析并为你的应用选择一种架构
- 决定使用哪些技术、编程语言和工具
- 考虑命名和编码规范
- 创建网站的基本结构并设置数据库
- 在网站骨架中实现错误处理例程和报告例程
