# 第 14 章：XML、RSS、WDDX 与 SOAP

通过使用 wsdl 文件，客户端代码变得更像“普通”的 PHP 代码。加载 wsdl 文件后，`SoapClient` 会为其中定义的每个方法创建魔术函数，现在可以直接调用这些函数。

如果服务端定义了 wsdl 文件中未包含的其他方法，仍然可以调用这些方法，但只能使用 `__soapCall()` 方法。

soap 函数可以通过 HTTPS 使用，以提高请求的安全性。此外，如果传输敏感数据，建议客户端使用额外参数传递凭证。这可以像其他参数一样实现，在客户端发起调用之前通过其他方式交换凭证值，服务端会在返回值之前验证凭证。这样一来，服务端可以通过使凭证失效来撤销客户端的访问权限。
