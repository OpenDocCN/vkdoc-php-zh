# 1. 什么是 PHP——我为什么要在意它？

PHP 常被批评者描述为最糟糕的编程语言之一，但它也是最流行的语言之一。它曾为 Google、Yahoo 和 Wikipedia 等网站提供支持。根据 Web Technology Surveys（[`https://w3techs.com/technologies/details/pl-php/all/all`](https://w3techs.com/technologies/details/pl-php/all/all)）的数据，在每五个使用服务器端语言的网站中，就有超过四个部署了 PHP。所以，PHP 显然有其可取之处。确实如此。PHP 是专门为构建动态网站而设计的。它是免费的；学习曲线非常平缓；而本书将向你展示如何将其付诸实践。

针对 PHP 的大多数批评都可以忽略。由于该语言在早期的发展方式，相关函数的名称和参数的顺序有时不一致。但是，借助一个好的编辑程序（参见本章“选择 PHP 编辑器时要注意什么”中的一些建议），这些问题很容易克服。它的一些特性在缺乏经验的人手中也会带来安全风险；但其中大部分已在本书所基于的 PHP 7 版本中消除了。

PHP 是一种成熟、强大的语言，已成为创建动态网站最广泛使用的技术。它通过以下方式让网站充满活力：

-   将网站反馈直接发送到你的邮箱
-   通过网页上传文件
-   从较大的图片生成缩略图
-   读取和写入文件
-   动态显示和更新信息
-   使用数据库显示和存储信息
-   使网站可搜索
-   以及更多……

通过阅读本书，你将能够做到所有这些。PHP 不仅易于学习；而且它是平台无关的，因此相同的代码可以在 Windows、macOS 和 Linux 上运行。使用 PHP 进行开发所需的所有软件都是开源且免费的。
