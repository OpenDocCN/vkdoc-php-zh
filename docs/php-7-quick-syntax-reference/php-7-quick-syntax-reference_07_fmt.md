# 4. 字符串

字符串是存储在变量中的一系列字符。在 PHP 中，字符串通常由单引号分隔。

```
$a = 'Hello';
```

## 字符串连接

PHP 有两个字符串运算符。点号被称为连接运算符（`.`）。它将两个字符串合并为一个。它还有一个配套的赋值运算符（`.=`），用于将右侧的字符串追加到左侧的字符串变量中。

```
$b = $a . ' World'; // Hello World
$a .= ' World';     // Hello World
```

## 字符串定界

PHP 字符串可以用四种不同的方式定界。有两种常见的表示法：双引号（`" "`）和单引号（`' '`）。它们的区别在于，单引号字符串中不会解析变量，而双引号字符串中会解析变量。

```
$c = 'World';
echo "Hello $c"; // "Hello World"
echo 'Hello $c'; // "Hello $c"
```

除非需要变量解析，否则通常倾向于使用单引号字符串，这突出了不会进行解析的特性。然而，双引号字符串被认为更易读，这使得选择更多取决于个人偏好。重要的是保持一致性。

除了单引号和双引号字符串之外，还有两种表示法：heredoc 和 nowdoc。这些表示法主要用于包含较大的文本块。

### Heredoc 字符串

heredoc 语法由`<<<`运算符后跟一个标识符和换行符组成。然后包含字符串，之后由包含该标识符的换行符来关闭字符串。在 heredoc 字符串中，变量会被解析，就像双引号字符串一样。

```
$s = <<<LABEL
Heredoc (with parsing)
LABEL;
```

### Nowdoc 字符串

nowdoc 字符串的语法与 heredoc 字符串相同，只是起始标识符需要用单引号括起来。在 nowdoc 字符串中，变量不会被解析。

```
$s = <<<'LABEL'
Nowdoc (without parsing)
LABEL;
```
