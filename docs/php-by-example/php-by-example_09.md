# 摘要

-   `if` 语句允许你根据某个条件的真假来执行不同的命令块。条件在 `if` 关键字后的括号中指定，后跟条件为真时要执行的命令。条件为假时执行的命令放在 `else` 块中。

-   `while` 循环语句允许你重复执行一个命令块。只要 `while` 关键字后指定的条件为真，就会执行这些命令。

-   `do-while` 语句与 `while` 语句的不同之处如下：在 `do-while` 语句中，先执行命令块，然后检查条件；而 `while` 语句的执行从检查条件开始。

-   `for` 循环语句的语法比 `while` 和 `do-while` 语句更复杂，但使用起来非常方便，其功能不逊于其他循环语句。

-   `break` 指令允许你终止循环语句的执行。`continue` 指令用于终止当前迭代（但不终止循环语句的执行）。

-   `switch` 语句允许你根据某个表达式的值执行不同的命令块。它包含带有控制值的 `case` 块。如果被检查的表达式的值与控制值匹配，则执行相应 `case` 块中的命令。如果未找到匹配项，则执行可选的 `default` 块中的命令。

-   `goto` 语句允许你跳转到程序中用标签标记的特定位置。标签在 `goto` 关键字后指定。要用标签标记程序中的某个位置，请将标签名称后跟冒号。

-   PHP 8 引入了 `match` 选择语句。该语句允许你检查某个表达式的值，并根据该值返回结果。从某种意义上说，`match` 选择语句可以解释为 `switch` 选择语句与三元运算符的共生体。
