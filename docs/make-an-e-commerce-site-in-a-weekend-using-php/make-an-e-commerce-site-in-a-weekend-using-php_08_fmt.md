# 总结

在本章中，你了解了所选商品是如何保存在`cart`表中，该表会记录访客的会话 ID。同时，你也看到了当访客进行操作时，购物车内容是如何被管理和修改的。你还了解了网站页眉如何显示购物车数量以及访客的登录状态。最后，你学会了如何提供配送信息、接受付款，并将所选商品保存到`orders`和`orders_details`表中。你还了解了不同类型的支付方式。

本书的目标是学习各种 PHP 语句和函数，以及如何将它们实际应用于构建一个功能齐全的电子商务网站。在本书中，你学习了如何将客户和商品的信息保存到 MySQL 数据库中，以便将来使用。你现在知道了下拉菜单是如何制作的，以及如何用它来链接网站的其他网页。

你了解了电子商务网站最关键和最重要的组成部分——购物车——是如何定义的，商品是如何在其中被选择的，以及它们是如何被操作的。你还学习了电子商务网站可用于接受客户付款的不同支付方式。

在本书中，我力求将代码写得尽可能简单，并为每个脚本提供了解释和输出结果。我希望你喜欢阅读本书，并且已经足够深入地掌握了这些概念，能够创建属于自己的功能齐全的电子商务网站。
