## 整数类型

整数是整数值。它们可以用十进制（基数为 10）、十六进制（基数为 16）、八进制（基数为 8）或二进制（基数为 2）表示法指定。十六进制数字前面带有`0x`，八进制数字前面带有`0`，二进制数字前面带有`0b`。

```php
$myInt = 1234; // 十进制数
$myInt = 0b10; // 二进制数（十进制 2）
$myInt = 0123; // 八进制数（十进制 83）
$myInt = 0x1A; // 十六进制数（十进制 26）
```

PHP 中的整数总是有符号的，因此可以存储正数和负数。整数的大小取决于系统字长，所以在 32 位系统上，最大可存储值是 2^(32-1)。如果 PHP 遇到更大的值，它会被解释为浮点数。

## 浮点数类型

浮点数类型可以存储实数。可以使用十进制或指数表示法来赋值。

```php
$myFloat = 1.234;
$myFloat = 3e2; // 3*10² = 300
```

浮点数的精度取决于平台。通常采用 64 位的 IEEE 格式，可以容纳大约 14 位十进制数字，最大十进制值为 1.8×10³⁰⁸。

## 布尔类型

布尔类型可以存储布尔值，布尔值只能是`true`或`false`。这些值使用`true`和`false`关键字来指定。

```php
$myBool = true;
```

## 空类型

不区分大小写的常量`null`用于表示没有值的变量。这样的变量被视为特殊的`null`数据类型。

```php
$myNull = null; // 变量被设置为 null
```

与其他值一样，`null`值根据变量使用的上下文会进行不同的求值。如果作为布尔值求值，它变为`false`；作为数字求值，它变为零（`0`）；作为字符串求值，它变为空字符串（`""`）。

```php
$myInt = $myNull + 0;      // 数字上下文（0）
$myBool = $myNull == true; // 布尔上下文（false）
echo $myNull;              // 字符串上下文（""）
```

### 默认值

在 PHP 中，可以使用尚未被赋值的变量。这些未定义的变量会被自动创建，并带有`null`值。

```php
echo $myUndefined; // 变量被设置为 null
```

虽然这种行为是允许的，但良好的编程实践是在使用变量之前定义它们，哪怕只是将变量设置为`null`。作为提醒，当使用未定义的变量时，PHP 会发出一个错误通知。根据 PHP 错误报告设置的不同，这条消息可能会或不会显示出来。

```
Notice: Undefined variable: myUndefined in C:\xampp\htdocs\mypage.php on line 10
```

## 运算符

运算符是一个符号，它使脚本执行特定的数学或逻辑操作。PHP 中的运算符可以分为五种类型：算术运算符、赋值运算符、比较运算符、逻辑运算符和位运算符。

### 算术运算符

算术运算符包括四种基本算术运算，以及用于获取除法余数的取模运算符（`%`）。

```php
$x = 4 + 2; // 6 // 加法
$x = 4 - 2; // 2 // 减法
$x = 4 * 2; // 8 // 乘法
$x = 4 / 2; // 2 // 除法
$x = 4 % 2; // 0 // 取模（除法余数）
```

幂运算符（`**`）是在 PHP 5.6 中引入的。它将左侧操作数提升到右侧操作数的幂。

```php
$x = 4 ** 2; // 16 // 幂运算
```

### 赋值运算符

第二类是赋值运算符。最重要的是赋值运算符（`=`）本身，它将一个值赋给一个变量。

```php
$x = 1; // 赋值
```

#### 组合赋值运算符

赋值运算符和算术运算符的一个常见用法是对一个变量进行操作，然后将结果保存回同一个变量。这些操作可以使用组合赋值运算符来缩短。

```php
$x += 5; // $x = $x+5;
$x -= 5; // $x = $x-5;
$x *= 5; // $x = $x*5;
$x /= 5; // $x = $x/5;
$x %= 5; // $x = $x%5;
```

在 PHP 5.6 中增加的幂运算符也有一个简写形式的赋值运算符。

```php
$x **= 5; // $x = $x**5;
```

#### 递增和递减运算符

另一个常见的操作是将变量递增或递减 1。这可以通过递增（`++`）和递减（`--`）运算符来简化。

```php
$x++; // $x += 1;
$x--; // $x -= 1;
```

这两个运算符都可以用在变量的前面或后面。

```php
$x++; // 后递增
$x--; // 后递减
++$x; // 前递增
--$x; // 前递减
```

无论使用哪一种，对变量的结果都是一样的。区别在于：后置运算符在改变变量之前返回原始值，而前置运算符则先改变变量，然后返回值。

```php
$x = 5; $y = $x++; // $x=6, $y=5
$x = 5; $y = ++$x; // $x=6, $y=6
```

### 比较运算符

比较运算符比较两个值，并返回`true`或`false`。它们主要用于指定条件，即求值为`true`或`false`的表达式。

```php
$x = (2 == 3);  // false // 等于
$x = (2 != 3);  // true  // 不等于
$x = (2 <> 3);  // true  // 不等于（备选写法）
$x = (2 === 3); // false // 全等
$x = (2 !== 3); // true  // 不全等
$x = (2 > 3);   // false // 大于
$x = (2 >= 3);  // false // 大于或等于
$x = (2 <= 3);  // true  // 小于或等于
```

严格相等运算符`===`和`!==`用于比较类型和值。之所以需要它们，是因为常规的“等于”（`==`）和“不等于”（`!=`）运算符在比较操作数之前会自动进行类型转换。当不需要“等于”运算符的类型转换功能时，建议使用严格比较。

```php
$x = (1 ==  "1"); // true  （值相同）
$x = (1 === "1"); // false （类型不同）
```

PHP 7 增加了一个新的比较运算符，称为太空船运算符（`<=>`）。它比较两个值，如果两个值相等则返回 0；如果左侧的值更大则返回 1；如果右侧的值更大则返回-1。

```php
$x = 1 <=> 1; // 0 (1 == 1)
$x = 1 <=> 2; //-1 (1 < 2)
$x = 3 <=> 2; // 1 (3 > 2)
```

### 逻辑运算符

逻辑运算符通常与比较运算符一起使用。逻辑与（`&&`）在左右两侧都为`true`时求值为`true`，逻辑或（`||`）在左侧或右侧至少有一个为`true`时求值为`true`。用于反转布尔结果的是逻辑非（`!`）运算符。请注意，对于“逻辑与”和“逻辑或”，如果左侧已确定结果，则不会对运算符的右侧进行求值。

```php
$x = (true && false); // false // 逻辑与
$x = (true || false); // true  // 逻辑或
$x = !(true);         // false // 逻辑非
```

### 位运算符

位运算符可以对数字的二进制位进行操作。例如，异或运算符（`^`）将开启在运算符一侧设置、但不在两侧都设置的位。

```php
$x = 5 & 4;  // 101 & 100 = 100 (4) // 与
$x = 5 | 4;  // 101 | 100 = 101 (5) // 或
$x = 5 ^ 4;  // 101 ^ 100 = 001 (1) // 异或（排他性或）
$x = 4 << 1; // 100 << 1  = 1000 (8) // 左移
$x = 4 >> 1; // 100 >> 1  =  10 (2) // 右移
$x = ~4;     // ~00000100 = 11111011 (-5) // 取反
```

这些位运算符和算术运算符一样，都有简写形式的赋值运算符。

```php
$x=5; $x &= 4;  // 101 & 100 = 100 (4) // 与
$x=5; $x |= 4;  // 101 | 100 = 101 (5) // 或
$x=5; $x ^= 4;  // 101 ^ 100 = 001 (1) // 异或
$x=5; $x <<= 1; // 101 << 1  = 1010 (10) // 左移
$x=5; $x >>= 1; // 101 >> 1  =  10 (2) // 右移
```
