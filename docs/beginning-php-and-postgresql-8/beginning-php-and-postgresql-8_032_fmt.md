## PHP 的日期与时间库

即使是最简单的 PHP 应用程序，也往往至少会用到几个 PHP 的日期和时间相关函数。无论是验证日期、以特定格式格式化时间戳，还是将人类可读的日期值转换为其对应的时间戳，这些函数在处理原本相当复杂的任务时都能发挥巨大作用。

### `checkdate()`

`boolean checkdate (int month, int day, int year)`

虽然大多数读者都能清晰地记得小学时学过的“三十天的是九月”这首童谣¹，但恐怕很少有人（包括在座各位）能完整背诵它。

幸运的是，`checkdate()` 函数能很好地完成日期验证任务：如果由 `month`、`day` 和 `year` 指定的日期有效，则返回 `TRUE`，否则返回 `FALSE`。让我们看几个例子：

```php
echo checkdate(4, 31, 2005);
// 返回 false

echo checkdate(03, 29, 2004);
// 返回 true，因为 2004 年是闰年

echo checkdate(03, 29, 2005);
// 返回 false，因为 2005 年不是闰年
```

### `date()`

`string date (string format [, int timestamp])`

`date()` 函数根据 `format` 指定的指令，返回当前时间和/或日期的字符串表示形式。表 12-1 列出了几乎所有可用的 `date()` 格式参数。在此我们决定不包含 Swatch 互联网时间² 的参数，敬请谅解。

如果包含可选的 `timestamp` 参数（以 Unix 时间戳格式表示），`date()` 将根据该参数生成日期字符串。`timestamp` 参数必须符合 GNU 日期语法规则。如果未提供 `timestamp`，则会使用当前的 Unix 时间戳。

**表 12-1.** `date()` 函数的格式参数

| 参数 | 描述 | 示例 |
|------|------|------|
| `a` | 小写的上午和下午 | `am` 或 `pm` |
| `A` | 大写的上午和下午 | `AM` 或 `PM` |
| `d` | 月份中的第几天，带前导零 | `01` 到 `31` |
| `D` | 三字母文本表示的星期几 | `Mon` 到 `Sun` |
| `F` | 月份的完整文本表示 | `January` 到 `December` |
| `g` | 12 小时制的小时，不带前导零 | `1` 到 `12` |
| `G` | 24 小时制的小时，不带前导零 | `1` 到 `24` |
| `h` | 12 小时制的小时，带前导零 | `01` 到 `24` |
| `H` | 24 小时制的小时，带前导零 | `01` 到 `24` |
| `i` | 分钟，带前导零 | `01` 到 `60` |
| `I` | 夏令时 | `0` 表示否，`1` 表示是 |
| `j` | 月份中的第几天，不带前导零 | `1` 到 `31` |
| `l` | 星期几的文本表示 | `Monday` 到 `Sunday` |
| `L` | 闰年 | `0` 表示否，`1` 表示是 |
| `m` | 月份的数字表示，带前导零 | `01` 到 `12` |
| `M` | 三字母文本表示的月份 | `Jan` 到 `Dec` |
| `n` | 月份的数字表示，不带前导零 | `1` 到 `12` |
| `O` | 与格林威治标准时间（GMT）的差值 | `-0500` |
| `r` | 根据 RFC 2822 格式化的日期 | `Tue, 19 Apr 2005 22:37:00 -0500` |
| `s` | 秒，带前导零 | `01` 到 `59` |
| `S` | 日期的序数后缀 | `st`、`nd`、`rd`、`th` |
| `t` | 一个月的天数 | 28 到 31 |
| `T` | 执行机器的时区设置 | PST, MST, CST, EST 等 |
| `U` | 从 Unix 纪元开始的秒数 | |
| `w` | 星期几的数字表示 | 0 代表星期日，6 代表星期六 |
| `W` | ISO-8601 格式的年中的第几周 | 1 到 53 |
| `Y` | 年份的四位数字表示 | 1901 到 2038 (Unix)；1970 到 2038 (Windows) |
| `z` | 年中的第几天 | 0 到 365 |
| `Z` | 以秒为单位的时区偏移量 | –43200 到 43200 |

尽管常年使用 PHP，许多 PHP 程序员仍然需要查阅 PHP 文档来回忆表 12-1 中所列出的参数。

因此，虽然你可能无法仅通过查看几个例子就记住如何使用这个函数，我们还是来看几个例子，以便让你更清楚地理解 `date()` 到底能做什么。

第一个例子演示了 `date()` 最常见的用途之一，即简单地向浏览器输出一个标准日期：

```php
echo "Today is ".date("F d, Y");
// Today is April 27, 2005
```

下一个例子演示如何输出星期几：

```php
echo "Today is ".date("l");
// Today is Wednesday
```

我们来尝试一种更详细的当前日期表示方式：

```php
$weekday = date("l");
$daynumber = date("dS");
$monthyear = date("F Y");
printf("Today is %s the %s day of %s", $weekday, $daynumber, $monthyear);
```

这将返回以下输出：

```
Today is Wednesday the 27th day of April 2005
```

你可能会想把与参数无关的字符串直接插入到 `date()` 函数中，像这样：

```php
echo date("Today is l the ds day of F Y");
```

确实，在某些情况下这能奏效，但结果可能非常不可预测。例如，执行上述代码会产生：

```
EDTo27pm05 0351 Wednesday 3008e 2751 27pm05 of April 2005
```

然而，由于标点符号不会与任何参数冲突，你可以根据需要随意插入它们。例如，要将日期格式化为 mm-dd-yyyy，可以使用以下代码：

```php
echo date("m-d-Y");
// 04-26-2005
```

### 处理时间

`date()` 函数也可以产生与时间相关的值。我们来看几个例子，首先从简单地输出当前时间开始：

```php
echo "The time is ".date("h:i:s");
// The time is 07:44:53
```

但这是上午还是下午呢？只需添加 `a` 参数：

```php
echo "The time is ".date("h:i:sa");
// The time is 07:44:53pm
```

##### `getdate()`

`array getdate ([int *timestamp*])`

`getdate()` 函数返回一个由时间戳组件组成的关联数组。除非提供了 Unix 格式的时间戳，否则此函数根据当前日期和时间返回这些组件。总共返回 11 个数组元素，包括：

- `hours`：小时的数字表示。范围是 0 到 23。
- `mday`：月份中的第几天的数字表示。范围是 1 到 31。
- `minutes`：分钟的数字表示。范围是 0 到 59。
- `mon`：月份的数字表示。范围是 1 到 12。
- `month`：月份的完整文本表示，例如 July。
- `seconds`：秒的数字表示。范围是 0 到 59。
- `wday`：星期几的数字表示，例如 0 代表 Sunday。
- `weekday`：星期几的完整文本表示，例如 Friday。
- `yday`：一年中的第几天（数字偏移量）。范围是 0 到 365。
