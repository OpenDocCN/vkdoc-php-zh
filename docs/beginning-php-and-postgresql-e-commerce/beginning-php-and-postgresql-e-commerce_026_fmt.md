# 第一部分：第一阶段开发

## 第 1 章：启动电子商务网站

“电子商务”这个词在过去几年中经历了一场显著的衰落。仅仅拥有一个电子商务网站的想法就足以让许多商人满怀期待。但现在，光说“电子商务是未来——要么上线，要么关门”已经不够了。你现在需要有说服力、现实且具体的理由，才能将你的业务带到线上。

本书专注于编程及相关的学科，例如创建、访问和操作数据库。但在我们深入探讨之前，我们需要理解最初导致创建电子商务网站的商业决策。

如果你想在今天构建一个电子商务网站，你必须回答一些棘手的问题。

好消息是，这些问题确实有答案，我们将在本章中尝试回答它们：

-   这么多大型电子商务网站都失败了。在如今这个更加严峻的环境中，电子商务还能给我带来什么？
